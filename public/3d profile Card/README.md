## 📖 Description

A modern, responsive, and interactive **3D profile card generator** that lets you create customized profile cards with an instant live preview. Features a beautiful UI with smooth animations, 3D tilt effects, and multiple export options.

Perfect for creating professional LinkedIn posts, portfolio cards, social media bios, and more!

---

## ✨ Features

### 🎯 Core Features
- **Live Preview** - See changes instantly as you type
- **3D Tilt Effect** - Interactive card that responds to mouse movement
- **Image Upload** - Upload files or paste image URLs
- **Complete Profile** - Name, profession, bio, and profile picture
- **Social Links** - GitHub, LinkedIn, Twitter/X, and Instagram integration
- **Link Validation** - Automatic URL validation for all social links

### 🎨 Customization
- **Theme Color Picker** - Choose any color for your card
- **Dark/Light Mode** - Toggle between themes with one click
- **3 Layout Options**:
  - **Classic** - Traditional horizontal layout
  - **Compact** - Centered circular profile image
  - **Poster** - Full-bleed background image

### 💾 Export & Storage
- **Multi-format Download** - Export as PNG, JPG, or PDF
- **Auto-save** - LocalStorage preserves your data automatically
- **High Quality** - Scale 2x rendering for crisp exports

### 📱 Responsive Design
- Mobile-friendly interface
- Touch-optimized for tablets and phones
- Adaptive layouts for all screen sizes

## 🚀 Installation & Setup

### Option 1: Direct Browser Opening (Simplest)
```bash
1. Clone or download the repository
2. Navigate to: public/3d profile Card/
3. Double-click index.html to open in browser
```

### Option 2: Local Server (Recommended)
```bash
# Using Python
python -m http.server 8080

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8080
```

Then visit: `http://localhost:8080/public/3d%20profile%20Card/index.html`

## 📖 Usage Guide

### Step 1: Fill Your Profile
1. Enter your **Name** (e.g., "Krishna Bhati")
2. Add your **Profession** (e.g., "UI/UX Designer")
3. Write a short **Bio** (max 220 characters)

### Step 2: Add Profile Image
- **Upload**: Click "Upload image" to select a file from your device
- **URL**: Paste an image link in the "Image URL" field

### Step 3: Add Social Links
Fill in your social media usernames or full URLs:
- GitHub
- LinkedIn
- Twitter/X
- Instagram

*Tip: Just enter your username (e.g., "krishna123") - full URLs are auto-generated*

### Step 4: Customize
- **Theme Color**: Click the color picker to choose your brand color
- **Layout**: Select Classic, Compact, or Poster
- **Dark Mode**: Toggle the sun/moon icon

### Step 5: Preview & Export
- Move your mouse over the card to see the **3D tilt effect**
- Click **"Download card"** button
- Choose format: **PNG** (transparent), **JPG** (smaller), or **PDF** (print-ready)
