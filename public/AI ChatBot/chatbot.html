Get Your Gemini API Key

Lumix Chat uses Google's Gemini API for chat. Getting a free key takes under 2 minutes.

1
Go to Google AI Studio
aistudio.google.com/app/apikey
2
Click "Create API Key"
Sign in with your Google account if prompted
3
Copy your key

Enter Your Gemini API Key

Your key is stored only in your browser's local storage. It is never sent to any server other than Google's API.

Please enter a valid Gemini API key (starts with AIza)
