# 🛒 Amazon App 

## 🚀 Overview

Amazon App is a responsive web project built using **HTML, CSS, and JavaScript**.  
It replicates the look and feel of the Amazon homepage, including navigation, product categories, banners, and cart functionality.

This version also includes **Sign In / Sign Up authentication using LocalStorage**, simulating backend-like behavior for user accounts. It is designed as a **frontend-only application** for learning, practice, and portfolio showcase.

---

# ✨ Features

- ✅ Responsive Amazon-style homepage layout
- ✅ Navigation bar with search, account, and cart icons
- ✅ Product category sections with images and deals
- ✅ Sign In / Sign Up authentication using LocalStorage
- ✅ Add to Cart functionality with cart count badge
- ✅ Dark / Light theme toggle
- ✅ Sidebar navigation menu
- ✅ Hover effects and interactive UI elements
- ✅ Mobile-friendly responsive design

# 🔑 Authentication Flow

## Sign Up
- Creates a new account
- Stores user credentials in LocalStorage

## Sign In
- Validates credentials from LocalStorage
- Creates an active session

## Sign Out
- Clears current user session data
