# 🛒 Amazon Clone

## 🚀 Overview

Amazon Clone is a responsive e-commerce front-end inspired by the Amazon shopping platform, built using **HTML, CSS .

The project recreates key Amazon UI components including a navigation bar, search functionality, hero banner, category sections, and product displays. It provides users with a familiar shopping experience while demonstrating modern front-end development concepts.

---

## ✨ Features

✅ Amazon-inspired responsive user interface  
✅ Modern navigation bar with delivery location section  
✅ Search bar with category selection dropdown  
✅ Hero banner showcasing featured products and offers  
✅ Product category cards with images and descriptions  
✅ Responsive design for desktop, tablet, and mobile devices  
✅ Interactive hover effects and smooth UI transitions  
✅ Shopping cart icon and account navigation section  
✅ Clean and beginner-friendly implementation
