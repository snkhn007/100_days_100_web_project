# 🛒 Amazon Clone

## 🚀 Overview

Amazon Clone is a **static web project** built using **HTML and CSS**.  
It replicates the **homepage layout of Amazon**, including navigation, product categories, and footer sections, providing a responsive and visually appealing design.

---

## ✨ Features

✅ Amazon-style navigation bar with logo, search bar, and links  
✅ Hero banner section with promotional content  
✅ Product category boxes (Clothes, Electronics, Furniture, etc.)  
✅ Footer with grouped links (Customer Service, Payment Options, etc.)  
✅ Responsive design for desktop and mobile devices  
✅ Clean and modular CSS styling

---

## 🛠️ Technologies Used

- HTML5  
- CSS3
