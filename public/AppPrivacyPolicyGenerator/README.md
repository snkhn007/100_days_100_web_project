# App Privacy Policy Generator

A lightweight, client-side web application that helps developers instantly generate standard Privacy Policies for their mobile and web applications.

## Features
* 📝 **Customizable:** Generates clauses based on specific permissions (Location, Camera, Analytics, Ads, etc.).
* 🔄 **Multi-format Support:** Outputs in both Markdown (.md) and raw HTML formats.
* 📋 **One-Click Copy:** Easily copy the generated policy directly to your clipboard.
* ⚡ **Fast & Secure:** 100% client-side JavaScript. No data is sent to external servers.

## Tech Stack
* HTML5
* CSS3
* Vanilla JavaScript

## How to Run Locally
1. Clone this repository to your local machine.
2. Open the `ProjectName/` directory.
3. Open `index.html` in your favorite web browser. 
4. No build tools or local servers are required!
