# App Privacy Policy Generator

A lightweight, client-side web application that helps developers instantly generate privacy policies for their mobile and web applications.

## Features

- 📝 **Customizable policies:** Generates clauses based on selected permissions and data types such as Location, Camera, Analytics, Ads, and more.
- 🔄 **Multi-format support:** Exports policies in both Markdown and raw HTML formats.
- 📋 **One-click copy:** Copy the generated policy directly to your clipboard.
- ⚡ **Fast and secure:** 100% client-side JavaScript with no data sent to external servers.
- 📱 **Responsive UI:** Works smoothly across desktop and mobile devices.
- 🎯 **Easy to use:** Fill in app details, select the required options, and generate a policy in seconds.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run Locally

1. Clone this repository to your local machine.
2. Open the project folder.
3. Open `index.html` in your browser.
4. No build tools or local server are required.
