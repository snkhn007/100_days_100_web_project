# Bill Splitter

## 📖 Description

Bill Splitter is a simple browser tool for dividing a bill amount among multiple people.

## ✨ Features

- Accepts bill-sharing inputs
- Calculates each person's share
- Presents results in a clean centered interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
Bill-Splitter/
└── index.html
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates form input handling, arithmetic calculations, inline scripting, and compact UI styling.
