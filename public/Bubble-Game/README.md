# 🎮 Bubble Game

## 🚀 Overview

Bubble Game is an interactive number-matching game built with **HTML, CSS, and JavaScript**.

Players must pop the bubble that matches the target number before the timer runs out. The game tracks score, levels, and completed rounds, with theme customization and responsive design.

---

## ✨ Features

* ✅ Dynamic bubble grid with random numbers
* ✅ **Hit Target** — Match bubbles to the displayed number
* ✅ **Score Tracking** — +10 points per correct hit
* ✅ **Level Progression** — Timer resets and levels increment automatically
* ✅ **Completed Levels Counter** — Persistent tracking via Local Storage
* ✅ **Dark/Light Theme Toggle** for user preference
* ✅ **Pause/Resume Timer** for flexible gameplay
* ✅ **Interactive Feedback** — Bubble pop animation, wrong click shake, score flash
* ✅ **Responsive Layout** — Optimized for desktop, tablet, and mobile
* ✅ **Rules Modal** — Quick access to game instructions

## 🎮 How to Play

1. Click **▶ Start** to begin the game.
2. Match the bubble number with the **Hit Target** number.
3. Each correct hit awards **+10 points**.
4. New bubbles are generated after every correct selection.
5. Complete levels before the timer reaches zero.
6. Use **Pause/Resume** to control gameplay.
7. Toggle **🌙 / ☀️** to switch themes.
8. Open **📜 Rules** for quick instructions.

## 🎨 Customization

### Modify Bubble Count

Update values in `script.js`:

```javascript
const bubbleCount = 120;
```

### Customize Theme Colors

Edit variables in `style.css`:

```css
:root {
  --primary-color: #3498db;
  --background-color: #ffffff;
}
```
