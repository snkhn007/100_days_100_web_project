# 💎 Glass & Shadow UI Studio

An interactive visual styling designer built using semantic HTML5, modern vanilla CSS3, and JavaScript. Design glassmorphic cards, adjust shadows/borders, preview live backdrops, and instantly export production-ready CSS code.

## ✨ Features

- **Backdrop Blur & Opacity Control** — Adjust glass frosting variables dynamically
- **Realistic Floating Backdrop Blobs** — Soft glowing bubbles drift behind the glass to highlight transparency
- **Advanced Shadows Sliders** — Shift vertical offsets and blurs for ideal elevation states
- **One-Click CSS Copier** — Instantly copies production-ready clean CSS classes
- **Dynamic Cross-Browser Code Output** — Emits standard properties including `-webkit-` browser prefixes
- **Responsive Layout** — Dark styling grids adapt fluidly to desktop monitors, tablets, and phones

## 🛠️ Tech Stack

- HTML5 (Semantic landmarks & structural accessibility tags)
- CSS3 (Floating pulsing blobs, keyframes animations, CSS custom properties)
- JavaScript (ES6+, real-time canvas calculations, Clipboard copy engine)

## 📁 File Structure

```
CSSShadowGenerator/
├── index.html    # Studio artboard layout & selector range panels
├── style.css     # Dark ambient glowing themes & floating pulse animation classes
├── script.js     # Glass/shadow mathematical equations & clipboard listeners
└── README.md     # Installation & operations handbook
```

## 🚀 How to Use

1. Launch `index.html` in your browser.
2. Drag opacity and blur sliders to set the card's transparency texture.
3. Configure Y-offsets and shadow blurs to float the element over the neon blobs.
4. Click **Copy CSS Code** to capture the cross-browser styles block for your projects.
