# Medical Appointment Calendar

A modern, animated, and responsive web app for booking medical appointments at a clinic or hospital.

## Features

- **Animated Background:** Dynamic, color-shifting floating gradient spheres with a moving overlay for a premium look.
- **Handwriting Font for Headings:** Uses the 'Pacifico' Google Font for a friendly, welcoming feel.
- **Loading Spinner:** Overlay spinner for slow network scenarios and initial load.
- **Dark Mode:** Toggle between light and dark themes.
- **Service Selection:** Choose from common medical services (checkup, consultation, procedure, etc.).
- **Calendar UI:** Select available dates and time slots, with animated transitions and hover effects.
- **Booking Summary:** See your selected service, date, and time before confirming.
- **PDF Confirmation:** Download a professional PDF confirmation of your appointment.
- **Accessibility:** Keyboard navigation, focus states, and color contrast.
- **Responsive Design:** Works beautifully on desktop and mobile.

## How it Works

- Select a medical service, pick a date and time, and confirm your booking.
- The app prevents double-booking and disables unavailable slots.
- After booking, download a PDF confirmation for your records.
- All transitions and feedback are animated for a delightful user experience.

## Technologies Used
- HTML5, CSS3 (with custom properties and animations)
- JavaScript (ES6+)
- [Anime.js](https://animejs.com/) for UI animations
- [jsPDF](https://github.com/parallax/jsPDF) for PDF generation
- Google Fonts (Pacifico)
