# Candy Crush Game

## 📖 Description

Candy Crush Game is a browser-based match-style puzzle game using images, JavaScript logic, and custom styling.

## ✨ Features

- Displays a candy-themed game board
- Uses image assets for visual gameplay
- Handles game interactions with JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
Candy_Crush_Game/
├── images/
├── background.jpg
├── index.html
├── script.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates grid-based game UI, asset management, event handling, and browser game state updates.
