# ClimaCode 2.0

## 📖 Description

ClimaCode 2.0 is a weather-focused front-end project with a dedicated JavaScript file for app behavior and styled page layout.

## ✨ Features

- Provides a weather application interface
- Separates markup, styling, and JavaScript logic
- Includes responsive page styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
ClimaCode 2.0/
├── index.html
├── main.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates weather app UI structure, client-side scripting, DOM updates, and responsive styling.
