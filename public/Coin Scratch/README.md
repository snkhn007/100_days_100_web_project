# Coin Scratch ASMR

## Description

Coin Scratch ASMR is a premium ASMR-inspired interactive web project where users scratch a rusted metallic coating to reveal a luxurious reward coin underneath. The project is designed to create a calming and satisfying user experience through realistic scratch interaction, smooth reward animations, and procedural metallic scratching audio.

The interface focuses on minimal luxury aesthetics with soothing colors, smooth animations, and immersive interaction design.

---

## Features

- Realistic scratch-to-reveal interaction
- Premium 3D metallic coin design
- Rusted scratchable coating layer
- Procedural ASMR scratching audio
- Smooth heart and reward reveal animation
- Continuous thin scratch rendering
- Restore coin functionality
- Calming pastel luxury UI
- Responsive design for desktop and mobile
- Smooth transitions and animations

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Canvas API
- Web Audio API

## Usage

1. Open the project in your browser.
2. Move the cursor over the coin surface.
3. Click and drag to scratch the rusted coating.
4. Reveal the hidden reward beneath the surface.
5. Enjoy the synchronized ASMR scratching audio.
6. Click the "Restore Coin" button to reset the experience.

For the best experience, use headphones.
