# 🎮 Connect 4 Game

A modern and interactive **Connect 4 Game** built using **React** and **Framer Motion** with smooth falling animations, responsive gameplay, and an elegant user interface.

This project recreates the classic Connect 4 experience with modern frontend technologies and engaging visual effects.

---

##  Features

* 🎯 Classic Connect 4 gameplay
* 🔴🟡 Two-player turn-based system
* ✨ Smooth falling disc animations
* 🏆 Winner detection system
* 🔄 Restart game functionality
* 📱 Fully responsive design
* ⚡ Fast and lightweight React application
* 🎨 Modern UI with clean styling
* 🧠 Efficient game logic implementation

## 🎮 How to Play

1. Players take turns dropping discs into columns.
2. The disc falls to the lowest available position.
3. The first player to connect:

   * 4 horizontally
   * 4 vertically
   * 4 diagonally

   wins the game.
4. Use the **Restart Game** button to play again.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/connect4-game.git
```

Navigate to the project folder:

```bash
cd connect4-game
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
