# Competitive Programming Contest Tracker

A clean, responsive web application that fetches and displays upcoming competitive programming contests in real time. 

## Features
- Real time data fetching using the Codeforces API
- Displays contest name, start time, and duration
- Converts UNIX timestamps to local timezone automatically
- Responsive layout for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open `index.html` in any modern web browser.
2. Ensure you have an active internet connection so the API can fetch the latest contests.
3. Enjoy the project!
