# CPU Scheduler Web project
The CPU Scheduler Web Project is an interactive Operating Systems simulation tool designed to demonstrate and visualize the behavior of different CPU scheduling algorithms. The project allows users to input custom process details, run scheduling simulations, and analyze performance metrics through dynamic tables and Gantt chart visualizations.

It helps students and learners understand how process scheduling works internally in an operating system by providing real-time execution flow, waiting time calculations, turnaround time analysis, and CPU performance insights in a visually interactive environment.

## Features
- Supports 8 CPU Scheduling Algorithms:
  - FCFS (First Come First Serve)
  - SJF (Shortest Job First)
  - SRTF (Shortest Remaining Time First)
  - Round Robin
  - Priority Scheduling (Preemptive & Non-Preemptive)
  - HRRN (Highest Response Ratio Next)
  - Multilevel Queue Scheduling
- Interactive and responsive user interface
- Dynamic process configuration system

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone or download the repository
2. Navigate to the `CpuScheduler` project folder
3. Open the project using a local server (recommended):
   - Use VS Code Live Server extension
   - Or any local development server
4. Open `index.html` in the live server
5. Select a scheduling algorithm
6. Enter process details and run the simulation
