
# 🧩 Crossword Puzzle Game

## 🚀 Overview

Crossword Puzzle Game is a responsive and interactive web-based puzzle built using **HTML, CSS, and JavaScript**.  
It challenges users with multiple levels of crossword puzzles related to programming and technology.  
Players progress through **10 puzzles**, earning points and unlocking increasing difficulty until they reach the final trophy 🏆.

---

## ✨ Features

* ✅ Interactive crossword puzzle grid  
* ✅ Dynamic clue validation (live feedback on inputs)  
* ✅ Timer with play/pause/resume controls  
* ✅ Scoring system (10 points per puzzle, total 100 points)  
* ✅ Progressive difficulty across 10 puzzles  
* ✅ Automatic puzzle progression when solved  
* ✅ Trophy 🏆 celebration at the end  
* ✅ Responsive design for desktop and mobile  
* ✅ Dark/Light theme toggle  

---

## 🛠️ Technologies Used

* HTML5  
* CSS3  
* JavaScript (ES6)

## 🎮 Gameplay Flow

1. Puzzle 1 loads → Solve → +10 points
2. Next puzzle loads automatically
3. Continue until Puzzle 10 → Total 100 points
4. Show 🏆 trophy animation and victory message
5. Difficulty increases with longer words and trickier clues
