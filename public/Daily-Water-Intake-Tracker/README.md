# Daily Water Intake Tracker

## Description

A hydration dashboard that helps users set a daily water goal, log intake quickly, and monitor progress with persistent browser storage.

## Features

* Stores the daily goal and current intake in `localStorage`.
* Displays goal, intake, remaining water, and completion percentage.
* Includes quick-add buttons for 250 ml, 500 ml, and 1000 ml.
* Supports custom intake amounts.
* Provides a reset button for daily progress.
* Updates an accessible progress bar and completion message in real time.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts
* localStorage

## How to Run

1. Clone the repository.
2. Navigate to `public/Daily-Water-Intake-Tracker`.
3. Open `index.html` in a browser.
