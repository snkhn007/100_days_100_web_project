# Data Structures Visualizer

## Description

Data Structures Visualizer is an interactive web-based project designed to help beginners understand how different data structures work through smooth animations and real-time visual feedback. The project currently includes Array, Stack, and Queue visualizations with operations such as insertion, deletion, searching, traversal, sorting, push, pop, peek, enqueue, dequeue, front, and rear.

The visualizer focuses on providing an educational and beginner-friendly experience with animated transitions, operation highlights, overflow and underflow states, and explanatory definitions for important concepts.

## Features

- Interactive Array Visualization
- Stack Visualization with push, pop, and peek operations
- Queue Visualization with insert, delete, front, rear, and peek operations
- Smooth animated transitions and visual feedback
- Overflow and underflow animations
- Dynamic array sorting and searching animations
- Beginner-friendly definitions with typing animation effects
- Responsive and modern UI design
- Premium olive-themed calm interface
- Sticky visualization containers while scrolling
- Real-time size and capacity indicators

## Technologies Used

- HTML5
- CSS3
- JavaScript ES6

## Installation/Setup

1. Clone the repository.

2. Navigate to the project folder.

3. Open the project in your preferred code editor.

4. Run the project using Live Server or open `index.html` in the browser.

## Usage

- Select a data structure from the navigation section.
- Perform operations using the provided controls.
- Observe animations and visual changes to understand how each data structure behaves.
- Explore the learning section to understand concepts such as traversal, sorting, enqueue, dequeue, overflow, underflow, and more.
