# Dining Philosophers Simulation

An interactive visual simulation of the classic Operating Systems concurrency problem using HTML5, CSS3 transitions, and vanilla JavaScript.

## Features
- **Visual Table Layout:** Watch 5 philosophers transition between thinking, hungry, and eating states.
- **Resource Management:** Real-time visibility tracking for forks on the table (faded out when picked up).
- **Deadlock Visualization:** Switch on "Force Deadlock Mode" to induce a circular wait condition where all philosophers starve.
- **Asymmetric Solution:** Normal mode resolves deadlock by forcing the 5th philosopher to pick up forks in reverse order.

## Technologies Used
- HTML5
- CSS3 Animations & Transitions
- Vanilla JavaScript (Asynchronous timers)

## How to Run
1. Open `index.html` directly in any modern web browser.
2. Click **Start Simulation** to run the resource allocation routine.
