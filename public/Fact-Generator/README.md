# Random Fact Generator

A simple and interactive web application that generates interesting random facts using a public Facts API. Built with React and Tailwind CSS, the project provides a clean and responsive user interface for discovering fun and educational facts.

## Features

* Generate random facts with a single click
* Responsive and modern user interface
* Fetches real-time facts from an API
* Smooth user experience with attractive styling

## Technologies Used

* React.js
* Tailwind CSS
* JavaScript
* Vite
* Facts API

## How to Run

1. Clone the repository

```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project
```

2. Navigate to the project directory

```bash
cd public/Fact-Generator
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open the local server URL shown in the terminal (usually `http://localhost:5173`)

6. Enjoy generating random facts!
