# Fake API Data Generator

A modern, fully responsive web application that generates realistic mock API and database data instantly. Built for developers to test frontend applications, UI demos, and database practice.

## Features

- **Premium UI**: Dark/light mode, glassmorphism, smooth animations, and responsive design.
- **Customizable Fields**: Choose exactly which fields you need (Name, Email, Phone, Address, Company, Age, etc.).
- **Multiple Formats**: Export data in JSON or CSV format.
- **Developer-Friendly Output**: View generated data in a syntax-highlighted code block.
- **Copy & Download**: Easily copy to clipboard or download as `.json` or `.csv` files.
- **Persistent State**: Remembers your preferences and theme using `localStorage`.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox/Grid, Glassmorphism, animations. No frameworks used.
- **Vanilla JavaScript**: DOM manipulation, random data generation, Clipboard API, and Blob API for downloads.

## How to Run Locally

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. No build tools or servers are required!
