# 🎨 Gradient Palette Generator

A beautiful, interactive CSS gradient generator built with pure HTML, CSS, and JavaScript.

## ✨ Features

- **Live Preview** — See your gradient update in real-time as you customize
- **Multiple Gradient Types** — Linear, Radial, and Conic gradients
- **Color Picker** — Add up to 5 color stops with intuitive color pickers
- **Angle Control** — Adjust gradient angle with a smooth slider
- **Random Generation** — Generate random gradients with one click
- **Copy CSS** — Copy the generated CSS code to your clipboard instantly
- **Save Favorites** — Save your favorite gradients to localStorage
- **Keyboard Shortcuts** — Space (randomize), C (copy), S (save)
- **Responsive Design** — Works on desktop, tablet, and mobile

## 🛠️ Technologies Used

- HTML5 (semantic elements)
- CSS3 (custom properties, glassmorphism, animations)
- Vanilla JavaScript (ES6+, no frameworks)

## 🚀 How to Use

1. Open `index.html` in your browser
2. Choose a gradient type (Linear, Radial, or Conic)
3. Pick colors using the color pickers
4. Adjust the angle with the slider
5. Click **Copy CSS** to copy the gradient code
6. Click **Save** to save your favorite gradients
