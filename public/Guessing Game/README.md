# 🎯 Number Guessing Game

An interactive and modern number guessing game built with HTML, CSS, and JavaScript.  
Features include difficulty levels, smart hints, analytics tracking, theme switching, and engaging animations.

---

## ✨ Features

- 🎮 **3 Difficulty Levels**
  - Easy: 1–50
  - Medium: 1–100
  - Hard: 1–200

- 🔢 **Attempt System**
  - Maximum 10 attempts per game
  - Tracks remaining tries dynamically

- 🧠 **Smart Hint System**
  - Hot / Warm / Cold feedback based on closeness
  - Directional hints (Higher / Lower)

- 📜 **Guess History**
  - Stores and displays all previous guesses

- 🏆 **Best Score Tracking**
  - Saved using `localStorage`
  - Persists across sessions

- 📊 **Game Analytics**
  - Total games played
  - Total games won
  - Win rate percentage

- 🌗 **Light / Dark Theme**
  - Toggle between themes
  - Saves user preference in localStorage

- 🎉 **Confetti Win Animation**
  - Celebration effect on correct guess

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
- Canvas API (for confetti animation)

---

## 🎮 How to Play

1. Select your difficulty level
2. Enter a number guess
3. Follow hints (Hot/Warm/Cold)
4. Try to guess the number in minimum attempts
5. Track your stats and beat your best score!
