# 🎯 Number Guessing Game

An interactive number guessing game with difficulty levels, attempt tracking, and best score saving.

## Features
- **3 Difficulty Levels**: Easy (1-50), Medium (1-100), Hard (1-200)
- **Attempt Tracking**: Maximum 10 attempts
- **Smart Hints**: Temperature-based feedback (Hot/Warm/Cold)
- **Guess History**: Track all previous guesses
- **Best Score**: Saved using LocalStorage
- **Responsive Design**: Works on all devices

## Technologies
- HTML5
- CSS3
- JavaScript ES6+

## How to Play
1. Select difficulty
2. Enter your guess
3. Get hints and try again
4. Beat your best score!
