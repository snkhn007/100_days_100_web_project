# ⚡ Interactive Wizarding World Spellbook & House Sorter

A magical, interactive web application built with vanilla web technologies. This project fetches live data from the Harry Potter API and features an interactive "Spell Caster" that dynamically alters the UI using CSS animations and state changes based on the spells you cast.

## ✨ Features

* **Interactive Spell Caster:** Type known spells into the input to trigger visual changes on the screen:
    * `Lumos`: Toggles a glowing light-mode effect.
    * `Nox`: Reverts the screen to the default dark mode.
    * `Reducto`: Applies a CSS keyframe shake animation to the interface.
* **House Sorter:** Filter the wizarding world roster by their Hogwarts House (Gryffindor, Slytherin, Hufflepuff, Ravenclaw).
* **Character Search:** Dynamically search through characters by their name.
* **Responsive Grid:** A clean, responsive grid layout to display character cards with their images and house details.

## 🛠️ Technologies Used

* **HTML** 
* **CSS** 
* **JavaScript**
* **API:** [HP-API](https://hp-api.onrender.com/)
