# HTML & CSS Animation Gallery

## 🚀 Overview

HTML & CSS Animation Gallery is a modern collection of pure HTML and CSS animations designed for learning, inspiration, and direct use in web projects.

The project showcases a variety of animation techniques including loaders, text effects, buttons, UI components, fun animations, hover effects, and interactive CSS-only components. Users can preview animations, explore their source code, and easily copy the HTML and CSS for use in their own projects.

All animations are built using **HTML5** and **CSS3** without any JavaScript animation libraries, making the project lightweight, educational, and beginner-friendly.

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd HTML_CSS_Animation
```

3. Open `index.html` in your browser

No build tools, frameworks, or dependencies are required.
