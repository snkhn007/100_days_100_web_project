# HTML & CSS Animation

## Overview
This project showcases a collection of pure **HTML** and **CSS** animations that demonstrate various keyframe effects, transitions, and interactive styling techniques. Each animation is built without any JavaScript, focusing on modern CSS capabilities.

## Live Demo
You can view the live demo of all animations here:

https://html-css-animation-01.netlify.app/

## Project Structure
- `index.html` – Entry point that displays all animation demos.
- `styles/` – Contains CSS files for each animation.
- `assets/` – Optional images or icons used by some demos.

## How to Run Locally
1. Clone the repository.
2. Open `public/HTML_&_CSS_Animation/index.html` in a browser.
3. Explore the animations and view the source code for each effect.
