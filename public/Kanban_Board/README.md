# 🚀 Advanced Kanban Board

A modern, responsive, and feature-rich Kanban Board application built using HTML, CSS, and JavaScript. This project helps users efficiently organize tasks, manage multiple projects, track progress through subtasks, and visualize productivity using a built-in analytics dashboard.

---

# 📖 Overview

The Advanced Kanban Board transforms traditional task management into a complete productivity solution. Users can create multiple boards for different projects, break tasks into smaller subtasks, monitor progress automatically, and gain valuable insights through real-time analytics.

Whether you're managing personal goals, college assignments, open-source contributions, or work projects, this application provides a structured and visual workflow to improve organization and productivity.

---

# ✨ Features

## 📋 Task Management

- Create new tasks
- Delete tasks
- Drag-and-drop task cards
- Organize tasks across workflow stages:
  - To Do
  - In Progress
  - Review
  - Completed
- Responsive task cards with progress indicators

---

## 🗂️ Multiple Board Support

Users can manage multiple independent Kanban boards, making it easier to separate tasks across different projects.

### Supported Actions

- Create Boards
- Switch Between Boards
- Rename Boards
- Delete Boards
- Independent task storage per board

### Example Use Cases

- Personal Tasks
- College Assignments
- Open Source Contributions
- Internship Work
- Team Projects

---

## ✅ Subtasks & Progress Tracking

Large tasks can be broken into smaller actionable subtasks.

### Features

- Add unlimited subtasks
- Mark subtasks as completed
- Automatic progress calculation
- Visual progress bars
- Real-time completion percentage updates

### Example

Build Portfolio Website

✔ Design Homepage

✔ Create Navigation

⬜ Add Animations

⬜ Mobile Optimization

Progress: 50%

---

## 📊 Analytics Dashboard

The application includes a real-time dashboard that provides productivity insights.

### Dashboard Metrics

- Total Tasks
- Completed Tasks
- Tasks In Progress
- Pending Tasks
- Completion Percentage

This helps users quickly understand project status and overall productivity.

## 🌙 Dark & Light Mode

The application includes theme support for better user experience.

### Features

- Dark Mode
- Light Mode
- Theme Persistence using Local Storage

## 💾 Data Persistence

All application data is stored locally using Browser Local Storage.

### Stored Data Includes

- Boards
- Tasks
- Subtasks
- Progress Status
- Theme Preferences

Users can refresh or reopen the application without losing data.

# 🎯 Problem Statement

Traditional task management applications often become difficult to use when handling multiple projects and complex workflows.

### Common Limitations

- No separation between projects
- Lack of task breakdown functionality
- No progress tracking system
- No productivity insights or statistics

This Advanced Kanban Board solves these issues by introducing multiple project boards, subtasks, progress tracking, and analytics.

- Create an intuitive project management tool
- Support multiple project boards
- Improve task organization
- Enable progress tracking through subtasks
- Provide productivity analytics
- Maintain a responsive and modern UI
