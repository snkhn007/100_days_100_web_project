# 📌 Kanban Board Web Application

A modern and responsive Kanban Board application built using HTML, CSS, and JavaScript that helps users organize and manage tasks visually through a drag-and-drop workflow.

## 🚀 Overview

The Kanban Board Web Application provides an intuitive way to manage tasks by organizing them into different workflow stages. Users can create, edit, delete, and move tasks between columns such as **To Do**, **In Progress**, **Review**, and **Completed**.

This project demonstrates practical frontend development skills, including drag-and-drop functionality, local storage management, responsive design, and modern UI/UX principles.

---

## ✨ Features

### Core Features

* Create new tasks
* Edit existing tasks
* Delete tasks
* Drag-and-drop task cards
* Multiple workflow columns:

  * To Do
  * In Progress
  * Review
  * Completed
* Responsive design for desktop, tablet, and mobile devices

### Advanced Features

* Task priority labels (High, Medium, Low)
* Due date management
* Task categories/tags
* Search and filter tasks
* Progress tracking statistics
* Dark/Light theme toggle
* Local Storage support for data persistence
* Smooth animations and modern UI

## 🎯 Problem Statement

Traditional task lists become difficult to manage when handling multiple tasks simultaneously. Users often need a visual workflow that clearly shows task progress and priorities.

A Kanban Board solves this problem by organizing tasks into columns representing different stages of completion, making it easier to track progress, improve productivity, and manage workloads effectively.

---

## 🎯 Objectives

* Create a visual task management system.
* Allow users to organize tasks into workflow stages.
* Enable drag-and-drop functionality for task movement.
* Improve productivity through better task visualization.
* Store tasks locally without requiring a backend.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd KanbanBoard
```

3. Open `index.html` in your browser.

No installation or backend setup is required.
