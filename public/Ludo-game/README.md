# 🎲 Ludo 4 Player

A fully interactive **4-Player Ludo Game** built using **HTML, CSS, and JavaScript**. Play with friends locally or challenge computer-controlled opponents in a classic Ludo experience with modern UI, animations, sound effects, timers, and victory celebrations.

---

## ✨ Features

### 🎮 Game Modes

* 👥 **4 Players Mode**

  * Enter custom player names.
  * Play with up to 4 human players on the same device.

* 🤖 **Player vs Computer Mode**

  * Play against 3 AI opponents.
  * Computer automatically rolls dice and makes strategic moves.

### 🎲 Gameplay Features

* Realistic dice rolling animation.
* Automatic turn management.
* Extra turn on rolling **6**.
* Token capturing (cutting opponent tokens).
* Safe zones where tokens cannot be captured.
* Exact roll required to reach home.
* Win detection when all 4 tokens reach the destination.

### ⏳ Turn Timer

* 30-second countdown timer for every player.
* Automatic turn skip on timeout.

### 🎨 User Interface

* Responsive Ludo board.
* Color-coded player panels.
* Active turn highlighting.
* Moveable token glow effect.
* Live game status updates.
* Activity log panel.

### 🎉 Winning Experience

* Animated victory screen.
* Trophy celebration.
* Confetti effects.
* Winner announcement card.
* Play Again option.

### 🔊 Sound Effects

* Dice roll sound.
* Token movement sound.
* Token capture sound.
* Victory sound.

## 🎯 How to Play

### Starting the Game

1. Open the game.
2. Select a game mode:

   * 4 Players
   * Player vs Computers
3. Enter player names (for 4-player mode).
4. Click **Start the Game**.

### Rules

* Roll a **6** to bring a token out of the yard.
* Tokens move according to the dice value.
* Landing on an opponent's token sends it back to the yard.
* Safe cells protect tokens from being captured.
* A token must reach home using the exact dice value.
* First player to get all 4 tokens home wins.

## 🤖 AI Logic

The computer opponent:

* Automatically rolls dice.
* Evaluates possible moves.
* Prioritizes:

  * Capturing opponent tokens.
  * Advancing tokens further.
  * Strategic progression toward home.
