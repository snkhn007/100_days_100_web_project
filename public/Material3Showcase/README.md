# Material 3 Expressive UI Showcase

A modern, responsive, and fully functional UI component showcase inspired by Android's latest Material 3 Expressive design system. 

## ✨ Features
* **Material 3 Expressive Styling:** Implements the latest Google M3 CSS variable system, including playful border radii, distinct elevations, and rich color mappings.
* **Instant Live Search:** A powerful JavaScript search bar that filters components in real-time based on names and meta-tags.
* **Dark/Light Theme Engine:** A built-in theme toggler that respects system preferences and saves user choices in Local Storage.
* **Ready-to-Use Code:** Includes a robust starter pack of Buttons, Cards, Inputs, Chips, Navigation bars, and Toggles.

## 🚀 How to Add More Components (Reaching 100+)
This project is designed to scale easily! To add more components:
1. Open `index.html`.
2. Copy an existing `<div class="component-card">` block.
3. Update the `data-tags` attribute with relevant search keywords.
4. Modify the internal HTML classes to create variations (e.g., changing a `.m3-btn-filled` to an icon-only button).

## 🛠 Tech Stack
* HTML5
* CSS3 (Native CSS Variables for M3 Theming)
* Vanilla JavaScript
