# 🟩 Micro-Habit Pixel Tracker

## 🚀 Overview

**Micro-Habit Pixel Tracker** is a simple and interactive web application designed to help users build consistency through small daily habits. Inspired by GitHub's contribution graph, it provides a visual pixel grid where each day changes color based on the number of completed habits.

The application allows users to select any day of the year, mark their completed habits, and automatically saves progress using the browser's **Local Storage**.

---

# ✨ Features

## 📅 365-Day Habit Grid

* Interactive grid representing all **365 days** of the year.
* Click on any square to select a day.
* Visual representation of daily progress through different color intensity levels.

---

## ✅ Habit Tracking

Track your daily progress with built-in habits:

* 📚 Read 5 Pages
* 💧 Drink 3L Water
* 🏃 Exercise / Stretch

The more habits completed in a day, the darker the corresponding pixel becomes.

---

## 🎨 Visual Progress System

The tracker includes four progress levels:

| Level      | Status               |
| ---------- | -------------------- |
| 🟫 Level 0 | No habits completed  |
| 🟩 Level 1 | One habit completed  |
| 🟩 Level 2 | Two habits completed |
| 🟩 Level 3 | All habits completed |

This creates a yearly heatmap that makes consistency easy to visualize.

---

# 💾 Data Persistence

* All habit data is automatically stored in the browser using **Local Storage**.
* Your progress remains saved even after refreshing or reopening the application.
* No account creation or internet connection is required.

---

# 🎮 How to Use

1. Click on any day square in the pixel grid.
2. Select or unselect the habits you completed.
3. Watch the square update its color based on your progress.
4. Continue filling the grid every day to build consistency.
