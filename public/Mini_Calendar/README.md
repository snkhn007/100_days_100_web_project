# 📅 Mini Calendar

A modern and responsive Mini Calendar web application built using HTML, CSS, and JavaScript. The application dynamically displays the current date, day, month, year, and real-time clock in an elegant user interface.

---

## ✨ Features

- 📆 Displays current date, month, and year
- 🕒 Real-time digital clock
- 🎨 Clean and modern user interface
- 📱 Responsive design
- ⚡ Lightweight and fast

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd Mini_Calendar
```

3. Open `index.html` in your browser

---

## 📖 Usage

- Open the application in any modern web browser.
- The calendar automatically displays the current date.
- The clock updates in real time.
