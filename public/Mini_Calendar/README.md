# 📅 Mini Calendar

A beautiful and interactive Mini Calendar web application built using **HTML, CSS, and JavaScript**. It displays the current date, live clock, day progress, greeting messages, and an interactive monthly calendar with theme customization.

---

## ✨ Features

### 📆 Live Date Display
- Shows the current:
  - Month
  - Year
  - Day of the week
  - Date

### 🕒 Real-Time Clock
- Live digital clock updating every second.
- Interactive analog clock mode.
- Toggle between digital and analog clocks instantly.

### 🌞 Smart Greetings
Displays a greeting based on the current time:
- 🌙 Burning the midnight oil (12 AM – 5 AM)
- ☀️ Good morning (5 AM – 12 PM)
- 🌤️ Good afternoon (12 PM – 5 PM)
- 🌆 Good evening (5 PM – 9 PM)
- 🌃 Good night (9 PM – 12 AM)

### 📊 Day Progress Tracker
- Visual progress bar showing how much of the current day has passed.
- Updates in real-time.

### 🗓️ Interactive Mini Calendar
- View any month and year using previous/next navigation.
- Highlights the current day.
- Highlights weekends separately.
- Quick **Jump to Today** button.

### 🎨 Dark & Light Theme
- Switch between dark and light modes.
- Theme preference is saved using **LocalStorage**.

### 💾 Persistent User Preferences
- Remembers your selected theme even after refreshing or reopening the browser.

### 📱 Responsive Design
- Works smoothly on:
  - Desktop 💻
  - Tablet 📱
  - Mobile devices 📲

## ⚙️ How It Works

- JavaScript fetches the current date and time using the `Date()` object.
- A timer updates the clock and day progress every second.
- The mini calendar dynamically generates dates for the selected month.
- Theme settings are stored in LocalStorage to maintain user preferences.
