# Mino - Modern Note Taking Platform 📝

Mino is a modern and elegant note-taking dashboard that helps users organize their ideas, tasks, and reminders in a clean and intuitive workspace. It provides folder management, note categorization, a calendar planner, archive and trash management, and local data persistence using the browser's Local Storage.

---

## ✨ Features

- 🗂️ Create and organize notes inside folders
- 📝 Add, edit, and manage personal notes
- 🎨 Customize folders and notes with color themes
- 🔍 Search notes and folders instantly
- 📅 Interactive calendar view with monthly navigation
- ⏰ Add reminder times and weekdays to notes
- 📌 Monthly checklist and task planner
- 🗃️ Archive notes without deleting them
- 🗑️ Move notes to trash and restore or permanently delete them
- 💾 Automatic data saving using Local Storage
- 📱 Fully responsive interface with mobile sidebar support
- 🌈 Quick theme accent color switching

## 🧩 Core Components

### 🗂️ Folder Management

- Create new folders
- Select custom folder colors
- Filter folders by week, month, or all time
- Delete folders while keeping notes unassigned

### 📝 Note Management

- Create and edit notes
- Assign notes to folders
- Add note colors and reminders
- View note details in a dedicated modal
- Search notes by title or content

### 📅 Calendar Planner

- Navigate through different months
- View notes associated with specific dates
- Create notes directly from calendar days
- Manage monthly checklist tasks

### 🗄️ Archive & Trash System

- Archive notes for later access
- Move unwanted notes to trash
- Restore deleted notes
- Permanently remove notes when no longer needed

---

## 💾 Data Storage

Mino stores all user data directly in the browser using **Local Storage**.

Stored data includes:

- Folder information
- Notes and their metadata
- Calendar checklist tasks
- User preferences and themes

No external database or server is required.
