# Morse Code Translator

An interactive Morse Code Translator that encodes text to Morse code and decodes Morse back to text in real-time, with built-in audio playback using the Web Audio API.

## Features

- Real-time text → Morse code encoding
- Morse code → text decoding mode
- Audio playback with adjustable speed (5–30 WPM)
- Copy output to clipboard
- Collapsible Morse reference chart (A–Z, 0–9, punctuation)
- Responsive design for mobile and desktop
- No dependencies — pure HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3 (glassmorphism, CSS animations)
- JavaScript ES6+ (Web Audio API)

## How to Run

1. Open `index.html` in a web browser
2. Enjoy the project!

## Usage

1. **Encode mode** (default): Type any text in the left panel — Morse code appears instantly on the right
2. **Decode mode**: Click "Morse → Text", enter Morse code (space between letters, `/` between words)
3. **Play audio**: Click "Play Audio" to hear the Morse code as beeps; adjust speed with the slider
4. **Copy**: Click the copy icon on the output panel to copy to clipboard
5. **Reference**: Click "Morse Reference" at the bottom to see the full character chart

## Morse Code Format (for decode mode)

- Separate letters with a **single space**: `... --- ...`
- Separate words with a **forward slash**: `.... . / .-- --- .-. .-.. -..`
