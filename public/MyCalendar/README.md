# MyCalendar 🎉📅

An **Interactive Festival & Event Calendar** built with **HTML, CSS, and JavaScript**.  
This mini project lets users plan festivals, holidays, and personal events with a clean UI and persistent notes using `localStorage`.

---

## ✨ Features
- 📆 Dynamic monthly calendar with navigation (previous/next month).
- 🖼️ Month-specific background images and theme colors.
- 📝 Add notes for each day with tags:
  - Normal Day
  - Festival / Holiday
  - Personal Event
- 💾 Notes saved in browser `localStorage` (persist after refresh).
- 🎨 Responsive design with smooth hover effects and active-day highlighting.
