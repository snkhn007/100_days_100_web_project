# Number Guessing Game

A simple and interactive Number Guessing Game built using HTML, CSS, and JavaScript. The player must guess a randomly generated number between 1 and 100 within 10 attempts.

## Features

- Random number generation between 1 and 100
- User input validation
- Feedback for each guess (too high / too low)
- Tracks previous guesses
- Displays remaining attempts
- Win and lose conditions
- Simple and responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your preferred web browser.
4. Start guessing the number!

## How to Play

1. Enter a number between 1 and 100.
2. Click **Submit Your Guess**.
3. The game will tell you whether your guess is too high or too low.
4. You have 10 attempts to find the correct number.
5. The game ends when you guess correctly or run out of attempts.
