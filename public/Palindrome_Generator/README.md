# 🔄 Advanced Palindrome Studio

An interactive and visually rich web application that allows users to **check, generate, and visualize palindromes** in real time. Built using **HTML, CSS, JavaScript, and Bootstrap**, the project provides an educational experience with animated character comparisons, theme customization, statistics tracking, and persistent history.

## 🚀 Features

### 🔍 Palindrome Detection

* Check whether a word or phrase is already a palindrome.
* Ignores spaces, punctuation, and letter casing during validation.
* Provides instant visual feedback for valid palindromes.

### ✨ Shortest Palindrome Generator

* Automatically generates the shortest possible palindrome by appending characters to the end of the input string.
* Displays the generated palindrome instantly.
* Supports words and phrases of different lengths.

### 📊 Interactive Visualization

* Step-by-step character comparison animation.
* Highlights:

  * Active characters being compared.
  * Matching character pairs.
  * Mismatched character pairs.
* Displays the number of comparisons performed.
* Demonstrates palindrome checking logic visually.

### 🎓 Educational Mode

* Toggle educational visualization mode.
* Helps users understand how palindrome algorithms work.
* Makes the application useful for students and beginners.

### 🎨 Multiple Themes

Choose between several beautiful themes:

* 🌙 Dark
* ☀️ Light
* 🌊 Ocean
* 🌅 Sunset
* 🌲 Forest

Theme preferences are automatically saved using browser storage.

### 📈 Statistics Dashboard

Tracks user activity, including:

* Total palindrome checks.
* Number of existing palindromes found.
* Number of generated palindromes.

Statistics are stored locally and remain available between sessions.

### 🕒 History Management

* Stores recent palindrome operations.
* Displays input, generated output, and timestamp.
* Keeps up to the latest 10 history records.
* Allows clearing the entire history.

### 📋 Copy & Utility Features

* Copy generated results directly to the clipboard.
* Clear input and reset visualization instantly.
* Press the **Enter key** to quickly start processing.

### 🎉 Interactive Effects

* Celebration confetti animation when a palindrome is successfully detected.
* Smooth UI animations and glassmorphism-inspired design.

## ⚙️ How It Works

1. Enter any word or phrase into the input field.
2. Click **Generate Palindrome** or press **Enter**.
3. The application will:

   * Check whether the input is already a palindrome.
   * Generate the shortest palindrome if necessary.
   * Animate the comparison process.
   * Update statistics and history.
4. Copy the result or clear the workspace anytime.
