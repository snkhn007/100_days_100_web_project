# 🌊 Particle Wave Animation

A visually engaging and interactive particle wave simulation built using **HTML5 Canvas**, **CSS3**, and **Vanilla JavaScript**. The project creates smooth animated particles that move in dynamic wave patterns, react to mouse/touch interaction, form particle connections, and support multiple customizable themes.

## 🚀 Features

### ✨ Interactive Particle System

* Smooth animated particle wave motion.
* Physics-inspired wave calculations.
* Randomized particle behavior for a natural appearance.
* Particle lifespan and automatic regeneration.

### 🎨 Multiple Color Themes

Switch between different visual styles:

* 🌊 Ocean
* 🔥 Fire
* 🌌 Aurora
* ⚫ Mono
* 🍬 Candy
* 🎲 Random Theme Generator

### 🎛️ Real-Time Controls

Adjust animation properties instantly:

* **Speed** — Control wave movement speed.
* **Particle Count** — Increase or decrease the number of particles.
* **Particle Size** — Change the size of particles.
* **Connection Range** — Control the distance for particle linking.

### 🖱️ Mouse & Touch Interaction

* Particles react to mouse movement.
* Touch support for mobile devices.
* Repulsion effect when the cursor approaches particles.
* Smooth interactive experience across devices.

### 🔗 Particle Connection Effects

* Dynamic lines are drawn between nearby particles.
* Connection opacity changes based on distance.
* Creates a modern network-style animation.

### ⏯️ Animation Controls

* Pause and resume the simulation.
* Reset particles instantly.
* Real-time FPS (Frames Per Second) counter for performance monitoring.

## ⚙️ Controls Overview

| Control          | Description                                     |
| ---------------- | ----------------------------------------------- |
| Speed Slider     | Changes the speed of particle movement          |
| Particles Slider | Controls the total number of particles          |
| Size Slider      | Adjusts particle radius                         |
| Connect Range    | Changes the distance at which particles connect |
| Theme Buttons    | Switch between different color themes           |
| Random Theme     | Applies a randomly selected theme               |
| Pause            | Stops and resumes animation                     |
| Reset            | Recreates the particle field                    |
