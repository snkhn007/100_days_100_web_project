# Perfect Slice

A fast paced, HTML Canvas arcade game where the player must navigate the board and draw lines while avoiding bouncing obstacles. 

## Features
- Real time HTML Canvas rendering.
- Path tracking and line drawing mechanics.
- Bouncing physics for enemy obstacles.
- Score tracking and lives system.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open `index.html` in any modern web browser.
2. Use the `W, A, S, D` keys or Arrow Keys to move the player (the white square).
3. Try to draw lines across the board without letting the red bouncing obstacles touch you or the line you are currently drawing!
