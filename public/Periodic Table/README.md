# 🧪 Interactive Periodic Table

An interactive and educational web application that visualizes the complete Periodic Table of Elements with advanced atomic models, trend analysis, search functionality, category filtering, and PDF export capabilities.

## 🚀 Features

### 📊 Complete Periodic Table

* Displays all 118 known chemical elements.
* Accurate positioning based on groups and periods.
* Separate Lanthanoid and Actinide series sections.
* Color-coded element categories.

### 🔍 Smart Search

* Search elements instantly by:

  * Element name
  * Chemical symbol

### 🎨 Category Filtering

Filter elements by:

* Alkali Metals
* Alkaline Earth Metals
* Transition Metals
* Post-transition Metals
* Metalloids
* Non-metals
* Halogens
* Noble Gases
* Lanthanoids
* Actinides

### ⚛️ Interactive Atomic Visualization

Click any element to open a detailed modal containing:

* Atomic Number
* Atomic Mass
* Proton Count
* Neutron Count
* Electron Count
* Electron Shell Configuration

Includes a dynamic animated **Bohr Model** featuring:

* Orbiting electrons
* Nucleus simulation
* Adjustable animation speed
* Orbit visibility toggle
* Electron trail effects
* Play/Pause controls

### 📈 Periodic Trend Analysis

Visualize important periodic properties:

* Atomic Mass
* Electronegativity
* Atomic Radius
* Ionization Energy

Features:

* Gradient heat-map visualization
* Highest and lowest value highlighting
* Scientific explanations of trends
* Period and group behavior analysis
* Direct Wikipedia reference links

### 📄 Export to PDF

Generate a high-quality PDF containing:

* Entire periodic table
* Current trend visualization
* Lanthanoid and Actinide series
* Proper multi-page support for large layouts

## ▶️ Getting Started

### 1. Clone the Repository


Navigate to the project directory:


### 3. Run Locally

Simply open:


in your preferred web browser.

No build tools or installation required.
