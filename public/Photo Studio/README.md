# 📸 Photo Studio

## 🚀 Overview
Photo Studio is a responsive portfolio project designed for showcasing professional photography.  
Built with **HTML and CSS**, it highlights elegant layouts, image presentation, and clean design principles.  
This project demonstrates how to structure a portfolio site with modern front‑end techniques.

---

## ✨ Features
- ✅ Responsive layout for desktop, tablet, and mobile  
- ✅ Hero section with featured photography  
- ✅ Gallery grid showcasing multiple photo categories  
- ✅ Smooth hover effects and transitions  
- ✅ Minimal, beginner‑friendly implementation  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3
