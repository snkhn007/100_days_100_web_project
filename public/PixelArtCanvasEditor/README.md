# 👾 Pixel Art Canvas Designer

A fully responsive, interactive pixel art studio built entirely with semantic HTML5, modern vanilla CSS3, and JavaScript. Paint grids, pick swatches, clear cells, and export pixel-perfect PNGs.

## ✨ Features

- **Grid Sizing Selector** — Switch instantly between 8x8, 16x16, or 32x32 dimensions
- **Pen & Eraser Tools** — Toggle painting modes smoothly
- **Drag Painting Support** — Click and sweep across cells to paint fluidly
- **Quick Swatches Palette** — Choose pre-defined colors or select a custom hex picker
- **Toggle Gridlines** — Easily turn cell outlines on/off
- **Sharp PNG Export** — Downloads pixel-perfect rendering maps with crisp edges
- **Fully Responsive UI** — Dark theme layout scaled beautifully to fit any screens

## 🛠️ Tech Stack

- HTML5 (Semantic elements & accessible grid structures)
- CSS3 (Frosted-glass UI cards, grid/flex layouts, visual checkerboards)
- JavaScript (ES6+, Event Listeners, HTML5 Canvas API export)

## 📁 File Structure

```
PixelArtCanvasEditor/
├── index.html    # Artboard Layout & side tools selector panels
├── style.css     # Dark checkboard backdrop designs & button visual state styles
├── script.js     # Cell painting registers, toggles, & PNG rendering loops
└── README.md     # Installation & how-to-use documentations
```

## 🚀 How to Use

1. Launch `index.html` in your browser.
2. Select your desired grid size from the dropdown.
3. Click or drag to draw across the pixel canvas checkerboard.
4. Toggle borders as needed to preview your creation.
5. Click **Download as PNG** to save a crisp copy of your art.
