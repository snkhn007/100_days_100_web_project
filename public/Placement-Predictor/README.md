# Placement Predictor

A smart web app that predicts placement probability and expected salary package based on student skills and profile.

## Features

- Placement prediction percentage
- Expected package prediction
- Modern responsive UI
- State dropdown selection
- Skill-based evaluation
- Internship and communication analysis

## Tech Stack

- HTML
- CSS
- JavaScript

## How It Works

The app calculates a score using:
- CGPA
- DSA skills
- Projects
- Communication skills
- Internship experience
- Tech stack

Based on the score, it predicts:
- Placement probability
- Expected package
- Improvement suggestions
