# 🎓 Placement Predictor

A modern, responsive, and interactive web application that helps students evaluate their placement readiness by analyzing academic performance, technical expertise, project experience, internships, and communication skills.

Placement Predictor generates placement probability estimates, expected salary packages, personalized improvement recommendations, and skill analytics through an intuitive dashboard designed to help students make informed career decisions.

# 🔍 Prediction Factors

The prediction engine evaluates multiple parameters to estimate placement readiness:

* 📚 CGPA
* 💻 DSA Skill Level
* 🛠 Number of Projects Completed
* 🗣 Communication Skills
* 🏢 Internship Experience
* 🚀 Preferred Technology Stack

These factors are combined to generate a placement probability score and estimated salary package.
