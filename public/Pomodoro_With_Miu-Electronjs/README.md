# Pomodoro With Miu 🐾

A cute cat-themed Pomodoro timer built with Electron.

## Features
- 25–5 Pomodoro cycle (focus + break)
- Sound notifications
- Distraction-free interface

## Tech Stack
HTML, CSS, JavaScript, Electron (desktop version)
