# Productivity Dashboard

A responsive productivity dashboard built using HTML, CSS, and JavaScript to help users manage daily tasks and stay organized.

## Features
- Today Task Manager
- Tomorrow Task Planner
- Progress Tracker
- Daily Streak Counter
- Dark/Light Mode Toggle
- Important Notes Section
- Local Storage Support
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Navigate to the Productivity-Dashboard folder
3. Open `index.html` in a web browser
