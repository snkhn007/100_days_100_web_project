# 🚀 Productivity Dashboard

A modern and responsive **Productivity Dashboard** built using **HTML, CSS, and JavaScript**. This application helps users organize daily tasks, track productivity progress, maintain notes, and monitor streaks—all within a clean and intuitive interface.

---

## 📖 Overview

The Productivity Dashboard is designed to improve daily organization and productivity. Users can manage tasks for today and tomorrow, track completion progress, save important notes, and switch between light and dark themes for a personalized experience.

---

## ✨ Features

| Feature                  | Description                                            |
| ------------------------ | ------------------------------------------------------ |
| ✅ Today Task Manager     | Add, complete, and manage tasks for the current day    |
| 📅 Tomorrow Planner      | Plan and organize tasks for the next day               |
| 📊 Progress Tracker      | Visual progress bar showing task completion percentage |
| 🔥 Daily Streak Counter  | Tracks productivity streaks to encourage consistency   |
| 📝 Notes Section         | Save important notes and reminders                     |
| 🌙 Dark Mode             | Toggle between light and dark themes                   |
| 💾 Local Storage Support | Automatically saves tasks and notes in the browser     |
| 📱 Responsive Design     | Optimized for desktop, tablet, and mobile devices      |

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser such as:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

### Installation

1. Clone the repository

```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project.git
```

2. Navigate to the project folder

```bash
cd Productivity-Dashboard
```

3. Open `index.html` in your browser
