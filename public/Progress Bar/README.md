Progress Bar 📊

A responsive and customizable Progress Bar component built using HTML, CSS, and JavaScript. This project demonstrates dynamic progress tracking with smooth animations and interactive controls, making it suitable for dashboards, loaders, task trackers, and UI-based applications.

📌 Features

Responsive progress bar UI
Smooth animation effects
Dynamic percentage updates
Interactive controls
Lightweight and fast
Beginner-friendly structure
Pure HTML, CSS, and JavaScript
Easy customization


🛠️ Tech Stack

| Technology | Usage |

| CSS3       | Styling & Animations |
