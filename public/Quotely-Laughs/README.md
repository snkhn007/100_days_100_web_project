# 😂 Quotely Laughs

A modern and responsive web application that generates **motivational quotes** and **funny jokes** instantly. Built with HTML, CSS, and JavaScript, featuring a beautiful glassmorphism UI, dark/light theme support, favorites, sharing, and history tracking.

![Quotely Laughs Banner](https://via.placeholder.com/1200x400?text=Quotely+Laughs)

---

## ✨ Features

### 🎯 Quote & Joke Generator

* Generate inspirational quotes instantly.
* Get funny jokes to brighten your day.
* Keyword-based generation support.

### 🌙 Dark / Light Theme

* Toggle between dark and light modes.
* Smooth theme transitions.

### ❤️ Favorites

* Save your favorite quotes.
* Save your favorite jokes.

### 📋 Copy to Clipboard

* Copy quotes with one click.
* Copy jokes instantly.

### 🐦 Social Sharing

* Share quotes on X (Twitter).
* Share jokes with friends.

### 📜 History Panel

* Track previously generated content.
* Easy access to past quotes and jokes.

### 📊 Generation Counter

* Displays total generated quotes count.

### 📱 Fully Responsive

* Mobile-friendly design.
* Optimized for tablets and desktops.
