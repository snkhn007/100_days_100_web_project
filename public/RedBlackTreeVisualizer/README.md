# Red-Black Tree Visualizer

An interactive Data Structures and Algorithms (DSA) tool that visualizes the construction of a Red-Black Tree.

## Features
- Real-time visualization of node insertions.
- Strictly adheres to all Red-Black Tree rules (Root is black, No adjacent red nodes, Black-height property).
- Animated Rotations (Left and Right rotations).
- Animated Recoloring phases to demonstrate self-balancing mechanics.
- Adjustable animation speed.
- Operation log detailing every rotation and recoloring step.

## Technologies Used
- HTML5
- CSS3 (Glassmorphism UI, CSS Variables, Flexbox, SVG)
- JavaScript (ES6 Classes, Async/Await Animation Queue)

## How to Run
1. Open `index.html` in a web browser.
2. Enter a number and click "Insert Node", or use the "Random" button.
3. Watch the tree balance itself automatically.
