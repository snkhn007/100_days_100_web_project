# RouteMaster – Campus Navigation

RouteMaster is a web-based campus navigation system that helps students and visitors easily locate buildings, facilities, hostels, and important destinations within the Amity University campus.

## Features

* Interactive campus map
* Smart location search
* Quick route suggestions
* Live occupancy information
* Campus zone navigation
* Recent searches history
* Modern responsive UI

## Technologies Used

* HTML
* CSS
* JavaScript
* SVG
* Font Awesome

## How to Run

1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Start exploring the campus and navigating to destinations.
