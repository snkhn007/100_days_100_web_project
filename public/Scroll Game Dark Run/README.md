# Scroll Game: Dark Run

## Overview

Scroll Game: Dark Run is a side-scrolling survival game where players navigate through a dark environment by scrolling or swiping. The goal is to avoid obstacles and reach the finish line safely.

## Features

* Horizontal scrolling gameplay
* Jump mechanic
* Obstacle collision detection
* Win and Game Over screens
* Timer and Best Time tracking
* Mobile touch support
* Restart functionality

## Gameplay

1. Scroll to start the game.
2. Navigate through the level while avoiding obstacles.
3. Reach the finish line to win.
4. Complete the run in the shortest time possible.

## Technologies Used

* HTML5
* CSS3
* JavaScript
