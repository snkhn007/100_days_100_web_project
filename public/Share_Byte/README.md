# ❤️ AaharShare – Food & Essentials Donation Platform

AaharShare is a community-driven donation platform designed to reduce food waste and help people in need. It allows users to donate surplus food, clothes, shoes, bags, blankets, and other essentials while ensuring that donations are renewed, reused, and delivered to the right people.

---

## 🌟 Features

### 🍲 Food Donation System
- Donate surplus food easily.
- Add donor details and food information.
- Set food expiry time.
- Live countdown showing remaining freshness time.
- Automatic removal of expired food listings.

### 📍 Location Integration
- Uses the **Geolocation API** to capture the donor's current location.
- Provides Google Maps links to help volunteers locate donations.
- Falls back to a default location when GPS access is unavailable.

### 💬 WhatsApp Claim System
- Volunteers or receivers can directly contact donors through WhatsApp to claim available food.

### 👕 Essentials Donation
Users can donate:
- Clothes
- Shoes
- Bags
- Blankets
- Other reusable household items

### 🌱 Renew & Reuse Initiative
AaharShare promotes sustainability by:
- Cleaning donated items.
- Restoring usable products.
- Delivering them to families and individuals in need.

### 📊 Donation Dashboard
- Displays active food donations.
- Shows freshness status using badges:
  - 🟢 Fresh
  - 🔴 Urgent (expiring soon)
- Updates automatically every minute.
