# Your Project Name
A fun and interactive memory game built using HTML, CSS, and JavaScript. The game challenges players to remember and repeat an increasingly long sequence of colors.

## Features
- 🎮 Classic Simon Says gameplay
- 🌈 Colorful interactive buttons
- 📈 Increasing difficulty with each level
- 💡 Visual feedback for button presses
- 🔄 Restart and replay anytime

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
Press any key to start the game.
Watch the color that flashes.
Click the same color button.
With each new level, a new color is added to the sequence.
Repeat the entire sequence correctly to advance.
If you click the wrong color, the game ends and your score is displayed.
Press any key to restart and try again.
