# ✨ Glowing Social Media Icons

A modern and visually stunning **Glowing Social Media Icons** web project built using **HTML** and **CSS**.
This project features animated neon-style social icons with a futuristic galaxy-inspired UI design.

---

## 🚀 Features

* 🌌 Futuristic galaxy background
* ✨ Neon glowing social media icons
* 🎨 Smooth hover animations
* 📱 Responsive design
* 🖤 Modern dark UI theme
* ⚡ Lightweight and fast

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project/tree/Main/public/Social%20Media%20Glowing.git
```

2. Open the project folder

```bash
cd Social-Media-Glowing
```

3. Run the project

Simply open `index.html` in your browser.
