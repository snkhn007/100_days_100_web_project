# Statistics Dashboard

A responsive statistics dashboard built using HTML, CSS, and JavaScript.

## Features

* Total Games Counter
* Player X Wins Counter
* Player O Wins Counter
* Draw Counter
* localStorage Persistence
* Reset Statistics Button
* Responsive Design

## Technologies Used

* HTML
* CSS
* JavaScript
* localStorage

## How to Run

1. Open `index.html` in a browser.
2. View and manage statistics.
3. Use the Reset button to clear stored data.
