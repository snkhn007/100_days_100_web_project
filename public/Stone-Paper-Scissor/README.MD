# 🎮 Rock Paper Scissors Game

A modern and interactive **Rock Paper Scissors** web game built using **HTML, CSS, and JavaScript** with beautiful UI effects, dark/light theme support, score tracking, animated screens, and customizable rounds.

---

## ✨ Features

- 🎨 Modern responsive UI
- 🌗 Dark / Light mode toggle
- 🧠 Play against computer AI
- 🔢 Custom number of rounds (1–10)
- 📊 Live scoreboard updates
- 🏆 Match winner screen
- 📋 Built-in game rules modal
- ⚡ Smooth animations and transitions
- 📱 Mobile-friendly design
- 🎯 Interactive gameplay experience

## 🎮 How to Play

1. Enter the number of rounds you want to play.
2. Click **Start Game**.
3. Choose:
   - ✊ Rock
   - ✋ Paper
   - ✌️ Scissors
4. The computer randomly selects its move.
5. Scores update automatically after every round.
6. After all rounds, the final winner is displayed.

---

## 📜 Rules

- ✊ Rock beats ✌️ Scissors
- ✌️ Scissors beats ✋ Paper
- ✋ Paper beats ✊ Rock
- Same choices result in a tie
- Highest score after all rounds wins the match
