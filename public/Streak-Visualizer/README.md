# LeetCode Streak Visualizer & Comparator 🔋⚔️

The **LeetCode Streak Visualizer** is an interactive web tool that allows users to fetch and visualize their LeetCode activity in real-time. It features an authentic LeetCode themed UI and dynamic battery visualization that charges based on your active daily streak!

## Features
* **Dual Modes:** * *Single Player:* Enter one username to view a detailed profile card and streak battery.
  * *VS Mode:* Enter two usernames to compare stats side-by-side. Winners are crowned dynamically! 👑
* **Real-Time Data:** Uses the `alfa-leetcode-api` to fetch live profile data, solved counts, and submission calendars.
* **Authentic UI:** Built using LeetCode's official dark mode color palette (including their signature Easy/Medium/Hard colors for the battery meter).
* **Smart Streak Calculation:** Parses raw UNIX timestamps from the submission calendar to calculate the user's exact current active daily streak.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Navigate to `/public/Streak-Visualizer/`
2. Open `index.html` in your preferred web browser.
3. Type a leetcode id into the search bar and hit enter!
