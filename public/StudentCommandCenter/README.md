## Student Command Center

A productivity dashboard designed to help students stay organized, focused, and productive.
The Student Command Center combines task management, note-taking, and a Pomodoro timer into a single, easy-to-use interface.

## Features

Theme Toggle
- Switch between dark mode and light mode.
- Improves accessibility and user experience.

Task Management
- Add new tasks.
- Delete completed or unwanted tasks.
- View the total number of active tasks.
- Tasks are automatically saved using Local Storage.

Notes Section
- Create and save study notes.
- Notes persist even after refreshing the page.
- Ideal for quick reminders, formulas, and revision material.

Pomodoro Timer
- 25-minute focus sessions.
- Reset timer functionality.
- Session completion notifications.
- Helps improve concentration and productivity.

## Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Browser Local Storage API

## How to Run

- Clone the repository:
git clone <repository-url>
- Navigate to the project folder:
cd Student-Command-Center
- Open index.html in your preferred web browser.
