# StudySync

## Description
StudySync is a modern and responsive educational landing page designed to promote seamless learning experiences. The project showcases an engaging user interface with interactive sections, smooth animations, testimonials, feature highlights, and newsletter subscription functionality. It is built using pure HTML, CSS, and JavaScript without any external frameworks.

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and clean educational landing page UI
- Interactive navigation menu with mobile support
- Sticky navbar with scroll effects
- Smooth scrolling navigation
- Animated hero section
- Feature cards with hover effects
- Customer testimonials section
- Newsletter subscription form
- Email validation with toast notifications
- Scroll-triggered animations using Intersection Observer
- Back-to-top button
- Social media integration
- Fully customizable design using CSS variables

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Playfair Display & DM Sans)

## Installation/Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/studysync.git
```

2. Navigate to the project folder:

```bash
cd studysync
```

3. Open the project in your preferred code editor.

4. Launch the application by opening:

```bash
index.html
```

or use VS Code Live Server for a better development experience.

---

## Usage

1. Open the website in a browser.
2. Explore the hero section and navigation menu.
3. View the platform features and advantages.
4. Read testimonials from users.
5. Subscribe using the newsletter form.
6. Use the navigation links or the Back-to-Top button for smooth browsing.
7. Test responsiveness across different screen sizes.
