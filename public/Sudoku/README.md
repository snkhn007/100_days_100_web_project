# Sudoku Game

A browser-based Sudoku puzzle game built using HTML, CSS, and JavaScript. The game generates playable Sudoku puzzles with multiple difficulty levels and provides an interactive experience through hints, notes mode, keyboard support, and real-time game tracking.

---

## Features

- Random Sudoku puzzle generation
- Easy, Medium, and Hard difficulty
- Interactive 9×9 Sudoku board
- Row, column, and box highlighting
- Notes mode
- Hint system
- Reset current game
- Timer
- Mistake tracking
- Keyboard support
- Responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd Sudoku-Game
```

Open

```bash
index.html
```

using any modern web browser.

---

## Usage

1. Select a difficulty.
2. Click **New Game**.
3. Fill empty cells.
4. Use Notes mode if required.
5. Use Hint when stuck.
6. Complete the puzzle before reaching the mistake limit.
