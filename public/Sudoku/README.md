# Sudoku Game

A browser-based Sudoku game where users can solve randomly generated puzzles using only HTML, CSS, and JavaScript.

## Features
- Generates a fresh Sudoku puzzle every game
- Easy, medium, and hard difficulty levels
- Interactive 9x9 board with row, column, box, and matching-number highlights
- Number pad, keyboard support, notes mode, hints, reset, timer, and mistake tracking
- Win and game-over messages
- Responsive layout for desktop and mobile screens

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open `index.html` in a web browser.
2. Choose a difficulty and solve the puzzle.
