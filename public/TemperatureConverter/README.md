# Temperature Converter
Convert the temprature in real time. 
## Overview
This is a simple web-based Temperature Converter that allows users to convert values between Celsius, Fahrenheit, and Kelvin in real-time.

## Features
- Convert temperature between Celsius,   Fahrenheit, and Kelvin
- Real-time conversion as user types
- Clean and responsive UI
- Input validation for empty or invalid values

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in any web browser

## How It Works
- When the user enters a value in any one input field, the other two fields are automatically updated
- JavaScript event listeners detect input changes and apply conversion formulas

## Conversion Formula
- Celsius to Fahrenheit: (C × 9/5) + 32
- Celsius to Kelvin: C + 273.15
- Fahrenheit to Celsius: (F − 32) × 5/9
- Kelvin to Celsius: K − 273.15

## Output
The application instantly displays converted temperature values based on user input.
