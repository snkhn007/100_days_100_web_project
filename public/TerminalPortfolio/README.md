# Terminal Portfolio

A realistic developer portfolio that looks and behaves like a modern terminal application. Built for the "100 Days 100 Web Projects" repository challenge.

## Features
- **Interactive Terminal UI:** Type commands to navigate the portfolio.
- **Boot Sequence:** Authentic typing animation upon loading.
- **Glassmorphism Design:** Modern cyberpunk aesthetic with blurred backgrounds and scanline effect.
- **Command History:** Use Up and Down arrows to cycle through previous commands.
- **Responsive:** Works seamlessly on mobile, tablet, and desktop.
- **Keyboard-only Navigation:** Optimized for power users with smooth focus management.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Flexbox, Grid, Animations, Custom Properties, Glassmorphism)
- Vanilla JavaScript (DOM manipulation, Async/Await, Event Handling)
- No external libraries or frameworks.

## Available Commands
- `help` - Shows all available commands.
- `about` - Displays developer bio.
- `skills` - Displays categorized skills with progress bars.
- `projects` - Shows a list of realistic portfolio projects.
- `contact` - Displays contact information with clickable links.
- `resume` - Simulates opening a resume.
- `github` - Simulates redirecting to GitHub.
- `linkedin` - Simulates redirecting to LinkedIn.
- `clear` - Clears the terminal history.
