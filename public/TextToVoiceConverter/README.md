# 🎤 Text & Voice Converter

A modern and responsive web application that converts **Text to Speech** and **Speech to Text** using the browser's built-in Web Speech API.

---

## ✨ Features

### 🔊 Text to Speech

* Convert typed text into speech
* Multiple language support:

  * English
  * Hindi
  * Spanish
* Start and stop speech controls

### 🎙️ Voice to Text

* Convert voice input into text in real time
* Continuous speech recognition
* Multiple language support
* Simple and responsive interface

---

## 🛠️ Technologies Used
