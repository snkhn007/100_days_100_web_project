# The Last Tab

A browser-native psychological experience — one tab, one article, and something lonely on the other end. The internet is shutting down. You're the last one here.

## Features
- Atmospheric narrative-driven experience
- Dynamic page degradation as you read
- Interactive chat with a non-human entity
- Multiple endings based on your choices
- Persistent state across visits
- Subtle glitch effects and environmental changes

## Technologies Used
- HTML
- CSS (animations, transitions, CSS variables)
- JavaScript (vanilla, localStorage)

## How to Run
1. Open index.html in a web browser
2. Wait for the page to load
3. Start reading — things will begin to change
