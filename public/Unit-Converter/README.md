# Unit Converter ⚡

A modern, responsive Unit Converter web application built using **HTML, CSS, and JavaScript**.  
It now includes **multi-theme support** and a **conversion history system**, making it more interactive and production-ready.

---

## 🚀 Features

- Convert between multiple unit categories
- Supports:
  - Length
  - Weight
  - Temperature
  - Time
  - Speed

- 🎨 Multiple Themes:
  - Aurora
  - Ocean
  - Emerald
  - Sunset
  - Midnight

- 🧾 Conversion History:
  - Stores last 10 conversions
  - Persistent using localStorage
  - Clear history option

- ⚡ Real-time conversion updates
- 🔁 Swap units instantly
- 💾 Data persistence using localStorage
- 📱 Fully responsive UI
- 🎯 Clean glassmorphism design

---

## 🧠 What This Project Solves

This project improves usability by:
- Adding personalization through themes
- Keeping track of past conversions
- Reducing repetitive input work
- Providing a smoother, more interactive UX
