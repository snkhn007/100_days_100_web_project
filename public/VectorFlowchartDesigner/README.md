# Vector Flowchart Designer

## Description
An advanced, browser-native vector diagramming and flowchart workspace. It utilizes native Pointer Capture APIs to provide fluid node dragging, continuous canvas panning/zooming, and real-time interactive coordinate tracking for bounding vector line connection paths.

## Features
- Drag-and-Drop Floating Node Spawn Array (Process, Decision, Terminator)
- Infinite Interactive Viewport Matrix Panning and Scroll-Wheel Zooming
- Real-Time Bending Vector Connector Lines with Target Anchor Snapping
- On-Canvas Text Editing Controls and Local Canvas Reset Handlers

## Technologies Used
- HTML5 (Semantic Structure, SVG Coordinate Spaces)
- CSS3 (Custom Variables, Workspace Blueprint Grids)
- JavaScript ES6 (Pointer Capture API, Matrix Transformations)

## How to Run
1. Open index.html in a modern web browser.
2. Drag or click shapes from the side panel tray to construct complex relational structures!
