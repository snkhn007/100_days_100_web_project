About This App

This Weather Forecasting App provides real-time weather updates for different cities. Users can search for weather information including temperature, humidity, wind speed, and sunrise/sunset timings in a clean and interactive interface.

Usage Guide

  • Enter a city name in the search bar
  • Click the Search button
  • View temperature, humidity, and wind details
  • Use the navigation bar for quick navigation
