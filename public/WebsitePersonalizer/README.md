# Interactive Website Personalizer

Answer five quick questions and watch a website rebuild itself to match your taste — live, in the browser, no server required.

## Features

- Card-by-card quiz with animated transitions
- Terminal-style "brewing" phase with typewriter output
- Live CSS-variable theming: vibe, color, mode, motion, typography
- 4 distinct vibes: Minimal · Cyberpunk · Aesthetic · Bold
- Light / Dark mode
- 4 color palettes: Violet · Rose · Emerald · Amber
- 4 motion profiles including reduced-motion option
- 4 typography scales
- Repersonalize button to re-run the quiz at any time
- No dependencies — pure HTML5, CSS custom properties, JavaScript ES6+

## Technologies Used

- HTML5
- CSS3 (custom properties, color-mix, backdrop-filter, keyframe animations)
- JavaScript ES6+ (DOM manipulation, async sequencing)

## How to Run

1. Open `index.html` in a web browser
2. Answer the five questions
3. Watch your personalized site appear!

## Quiz Questions

| # | Question | Options |
|---|----------|---------|
| 1 | Aesthetic vibe | Minimal · Cyberpunk · Aesthetic · Bold |
| 2 | Light or dark | Light · Dark |
| 3 | Signature color | Violet · Rose · Emerald · Amber |
| 4 | Motion preference | Fluid · Subtle · Slow · None |
| 5 | Reading style | Compact · Default · Relaxed · Wide |
