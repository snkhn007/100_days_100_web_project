# Age Calculator

A precise age calculator that calculates exact age in years, months, and days with additional features.

## Features

- Calculate exact age in years, months, and days
- Auto-fills today's date
- Next birthday countdown
- Zodiac sign display
- Total weeks and hours lived
- Responsive design for mobile
- Keyboard support (Enter key)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Use

1. Select your date of birth
2. Click "Calculate Age" or press Enter
3. View your exact age and additional info
