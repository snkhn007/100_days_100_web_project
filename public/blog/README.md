# Blog

## 📖 Description

Blog is a multi-page static blogging interface with landing and article pages styled through separate CSS files.

## ✨ Features

- Includes main and blog article pages
- Uses dedicated scripts for page interactions
- Organizes visual assets in an assets folder

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
blog/
├── assets/
├── blog-script.js
├── blog.html
├── main.html
├── script.js
├── style.css
└── styles.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `main.html` or `blog.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates multi-page static site organization, reusable styling, page-specific JavaScript, and blog content layout.
