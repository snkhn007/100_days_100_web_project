## Description

Bubble Game is a fun and interactive browser-based game built using HTML, CSS, and JavaScript. Players have to click on the target bubble before the timer runs out to increase their score. The game focuses on quick reflexes and provides an engaging UI with dynamic bubble generation and score tracking.

## Features
- Real-time score tracking
- Interactive bubble clicking gameplay
- Level wise system to track progress
- Dynamic bubble creation
- Responsive and clean UI
- Instant score updates

## Technologies Used
- HTML5
- CSS3
- JavaScript ES6

## Installation/Setup
- Clone the repository:
git clone <repository-link>
- Navigate to the project folder:
cd bubble-game
- Open index.html in your browser.

## Usage
- Start the game by opening the project in the browser.
- Click on the matching bubble and make combos of three to shoot the bubbles.
- Correct clicks increase your score.
- Try to achieve the highest score possible before time runs out.
