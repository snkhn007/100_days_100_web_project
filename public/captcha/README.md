# Captcha

## 📖 Description

Captcha is a modern, front-end verification demo with custom styling, JavaScript validation, and a comprehensive analytics dashboard!

## ✨ Features

- **Multiple CAPTCHA Types**: Text, Image, Audio, Math
- **Difficulty Levels**: Easy, Medium, Hard
- **Analytics Dashboard**:
  - Total Attempts
  - Successful/Failed Count
  - Success Rate
  - Streak Tracking
- **Achievements**: Beginner (5), Intermediate (20), Expert (50)
- **Recent Activity**: Last 5 attempts
- **Performance Insights**:
- **Responsive Two-Column Layout**:
- **Glassmorphism Design**:
- **LocalStorage Persistence**:
- **Reset Functionality**:
