# Captcha

## 📖 Description

Captcha is a front-end verification demo that displays a captcha challenge with custom styling and JavaScript validation.

## ✨ Features

- Shows a styled captcha verification screen
- Uses a background image for visual presentation
- Validates user input with JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
captcha/
├── captcha-bg.jpg
├── captcha.css
├── captcha.html
└── captcha.js
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `captcha.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates basic client-side validation, user input checking, and separation of HTML, CSS, and JavaScript files.

## 🔮 Future Enhancements
