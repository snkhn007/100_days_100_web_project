# Checkers Game

## 📖 Description

Checkers Game is a browser-based checkers implementation with player-versus-player and player-versus-bot modes.

## ✨ Features

- Supports player versus player mode
- Supports player versus bot mode
- Renders a playable checkers board in the browser

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
checkers game/
├── index.html
├── script.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates board game state management, click-based interactions, mode selection, and rule-driven UI updates.
