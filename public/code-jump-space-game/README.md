# Code Jump Space Game

## 📖 Description

Code Jump Space Game is a Doodle Jump-style browser game using platform and character assets with JavaScript gameplay logic.

## ✨ Features

- Displays a vertical jumping game scene
- Includes character, platform, pause, play, resume, and restart assets
- Handles gameplay controls and state through JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates canvas or DOM-based game loops, collision-style platform gameplay, asset loading, and game state controls.
