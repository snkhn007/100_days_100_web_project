# Code Playground

## 📖 Description

Code Playground is an interactive browser sandbox for writing and running code from a simple web interface.

## ✨ Features

- Provides editable code areas
- Includes a run button for executing code output
- Separates layout, styling, and logic across dedicated files

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
code-playground/
├── index.html
├── script.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates live preview interfaces, text input handling, iframe or output rendering patterns, and front-end tooling concepts.
