# Color Sort Puzzle

## 📖 Description

Color Sort Puzzle is a single-file browser puzzle game where colors are arranged and moved between containers.

## ✨ Features

- Presents a color sorting puzzle interface
- Uses embedded CSS and JavaScript in one HTML file
- Tracks puzzle interaction directly in the browser

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
color sort puzzle/
└── index.html
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates puzzle state logic, click interaction design, single-file web app organization, and visual UI feedback.
