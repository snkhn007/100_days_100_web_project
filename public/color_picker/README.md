# Color Picker

## 📖 Description

Color Picker is a custom browser tool for selecting, displaying, and saving color values.

## ✨ Features

- Provides an interactive color picker interface
- Includes a save color action
- Uses custom CSS for the picker layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
color_picker/
├── index.html
├── script.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates color input handling, DOM updates, UI styling, and small-tool interaction design.
