# Connect 4 Game

## 📖 Description

Connect 4 Game is a browser-based Connect Four implementation with player-versus-player and player-versus-bot modes.

## ✨ Features

- Supports player versus player mode
- Supports player versus bot mode
- Renders an interactive Connect Four board

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
connect 4 game/
├── index.html
├── script.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates grid-based game logic, turn management, win detection, and mode switching in JavaScript.
