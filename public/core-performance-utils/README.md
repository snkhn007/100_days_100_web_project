# Core Performance Utils

## 📖 Description

Core Performance Utils is a small JavaScript utility demo for measuring and demonstrating performance-related helper functions.

## ✨ Features

- Provides reusable performance utility functions
- Includes a demo script for usage examples
- Runs in a browser through a simple HTML page

## 🛠️ Technologies Used

- HTML5
- JavaScript

## 📂 Project Structure

```text
core-performance-utils/
├── demo.js
├── index.html
└── perfUtils.js
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates modular JavaScript utilities, performance measurement concepts, and demo-driven browser testing.
