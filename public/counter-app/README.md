# Counter App

## 📖 Description

Counter App is a simple interactive browser app for increasing, decreasing, and displaying a numeric counter.

## ✨ Features

- Displays the current counter value
- Provides buttons for counter interaction
- Updates the UI dynamically with JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
counter-app/
├── index.html
├── script.js
└── style.css
```

## 🚀 Getting Started

### Installation

No installation is required.

### Run

Open `index.html` in a browser.

## 🎯 Learning Objectives

This project demonstrates basic DOM selection, event listeners, state updates, and button-driven user interactions.
