# NovaConvert Currency Converter

## Description

NovaConvert is a polished currency converter that calculates live exchange values, supports quick currency pairs, and stores recent conversions in the browser.

## Features

* Converts amounts using live exchange rates from `api.exchangerate-api.com`.
* Includes selectable source and target currencies for common global currencies.
* Provides quick conversion shortcuts such as USD to INR and EUR to INR.
* Swaps selected currencies with one click.
* Saves up to six recent conversions in `localStorage`.
* Shows loading states, validation messages, reset behavior, and animated results.

## How to Run

1. Clone the repository.
2. Navigate to `public/currency_converter`.
3. Open `index.html` in a browser.
4. Ensure you have an internet connection for live exchange rates and CDN assets.
