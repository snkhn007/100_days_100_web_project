# 😂 Dad Jokes Generator

A simple and fun web application that fetches random dad jokes using the **icanhazdadjoke API** and displays them with a modern, responsive user interface.

## 🚀 Features

* Fetches random dad jokes from an external API
* One-click joke generation
* Responsive design for desktop and mobile devices
* Modern and clean UI
* Built using Vanilla JavaScript (no frameworks)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API

## 📂 Project Structure

```bash
Dad-Jokes-Generator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ⚙️ How It Works

1. The application sends a request to the Dad Jokes API.
2. The API returns a random joke in JSON format.
3. JavaScript extracts the joke and displays it on the page.
4. Clicking the **Get Another Joke** button fetches a new joke instantly.

## 🔧 Installation

1.Clone the repository:

```bash
git clone https://github.com/your-username/dad-jokes-generator.git
```

2.Navigate to the project directory:

```bash
cd dad-jokes-generator
```

3.Open `index.html` in your browser.
