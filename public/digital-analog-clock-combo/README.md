# ⏰ Digital & Analog Clock Combo

A modern and interactive clock application that combines a real-time Digital Clock and an elegant Analog Clock in a single interface. Built using HTML, CSS, and JavaScript, this project demonstrates time-based DOM manipulation, dynamic UI updates, and responsive design principles.

---

## ✨ Features

- 🕒 Real-time Digital Clock (HH:MM:SS)
- ⌚ Fully Functional Analog Clock
- 🔄 Automatic Updates Every Second
- 🎨 Modern and Responsive User Interface
- 📱 Mobile-Friendly Design
- ⚡ Lightweight and Fast Performance
- 🧩 Clean and Beginner-Friendly Code Structure

## 🎯 How It Works

### Digital Clock
- Fetches the current system time using JavaScript.
- Updates every second using `setInterval()`.

### Analog Clock
- Calculates the rotation angle of:
  - Hour Hand
  - Minute Hand
  - Second Hand
- Updates dynamically to reflect the current time.
