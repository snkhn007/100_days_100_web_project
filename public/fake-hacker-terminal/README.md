# ⚡ Fake Hacker Terminal

A stylish fake hacker terminal built using **HTML, CSS, and JavaScript**.
This project simulates a real command-line interface with interactive commands, cyberpunk aesthetics, and terminal-style animations.

---

## 🚀 Features

* Interactive terminal commands
* Neon cyberpunk UI
* Real-time command responses
* Fake hacking simulation
* Linux-style commands
* Smooth scrolling terminal
* Responsive design

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```bash
fake-hacker-terminal/
│
├── index.html
├── style.css
└── script.js
```

---

## 💻 Available Commands

| Command    | Description                 |
| ---------- | --------------------------- |
| `help`     | Show all commands           |
| `about`    | About the developer         |
| `skills`   | Display skills              |
| `projects` | Show projects               |
| `contact`  | Contact information         |
| `date`     | Current date and time       |
| `pwd`      | Show current directory      |
| `whoami`   | Display current user        |
| `ls`       | List files                  |
| `hack`     | Start fake hacking sequence |
| `clear`    | Clear terminal              |

---

## ⚙️ How To Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/fake-hacker-terminal.git
```

2. Open the project folder

3. Run `index.html` in your browser

OR

Use **Live Server** in VS Code for a better experience.
