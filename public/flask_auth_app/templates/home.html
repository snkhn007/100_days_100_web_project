Flask Authentication App

Modern auth, built with Python.

This demo uses Flask, SQLite, hashed passwords, and JWT cookies to manage sign up, login, protected content, and logout in a compact backend app.

  • Secure login flow Passwords are hashed before storage and validated on login.
  • JWT session cookie The app stores an HTTP-only token to keep the user signed in.
  • SQLite-backed users All users live in a local database created automatically on startup.
