# ⏱️ Focus Timer

A minimal and calming Pomodoro-style focus timer designed to help users stay productive while working or studying.

## 🌐 Live Demo

[View Live](https://focustimer-brown.vercel.app)

## 📸 About

The Focus Timer lets users set a custom work duration and break intervals. Motivational messages appear at regular intervals to keep users encouraged during long sessions.

## ✨ Features

- Custom focus timer (set your own minutes)
- Break intervals during focus sessions
- Automatic pause during breaks
- Motivational messages at regular intervals
- Minimal, distraction-free UI
- Responsive design for all devices

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- shadcn/ui
