# 🇮🇳 Indian Flag Animation Project

## 📌 Description
A UI-based web project featuring a waving Indian flag animation, inspirational quotes, and background music.

---

## ✨ Features

- 🇮🇳 Realistic CSS flag waving animation
- 🎨 Improved UI with background blur effect
- 🎵 Background music (Vande Mataram instrumental)
- 💬 Inspirational quote section with navigation
- 📱 Responsive design improvements

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox + Animations)
- JavaScript (DOM manipulation)
- Audio + Video background integration

---

## 🧪 Testing
- Tested on Chrome browser
- Responsive across multiple screen sizes

---

## 🚀 Improvements Made

- Animation added for realistic flag movement
- Background music integration
- Layout optimization for responsiveness
