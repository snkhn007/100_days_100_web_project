# Instagram Landing Page Clone

A lightweight, front‑end‑only clone of Instagram's landing page built with **HTML**, **CSS**, and **JavaScript**. The project runs entirely in the browser – no backend or authentication is required.

---

## ✨ Features
- Responsive layout matching Instagram's desktop UI
- Sidebar navigation (Home, Explore, Reels, Messages, Profile)
- Stories carousel with auto‑slide
- Image feed with like and comment placeholders
- Explore grid, Reels carousel, and DM inbox mock‑ups
- Dark‑mode ready colors and smooth micro‑animations
- Pure client‑side data model (`data.js`) for demo content

---

## 🛠️ Tech Stack
- **HTML5** – Semantic structure
- **CSS3** – Flexbox/Grid, custom properties, transitions, glass‑morphism style
- **JavaScript (ES6+)** – State handling, DOM manipulation, simple routing

## 🚀 Getting Started
1. **Clone or download** this repository.
2. Open a terminal and navigate to the project folder:
   ```bash
   cd C:\Users\HP\.antigravity\instagram-landing
   ```
3. Start a local static server (Python example):
   ```bash
   python -m http.server 8000
   ```
   *(You can also use `npx serve -l 8000` or any other static‑file server.)*
4. Open your browser and go to:
   ```
   http://localhost:8000
   ```

## 🎨 Design Notes
- Colors are defined using **HSL** variables for easy theming.
- Layout uses **CSS Grid** for the main feed and **Flexbox** for sidebar navigation.
- Micro‑animations (hover effects, story slide transitions) add a premium feel.

---

## 📦 Extending the Project
- Add real API calls to fetch posts from a backend.
- Persist likes/comments using `localStorage` or a simple server.
- Implement dark‑mode toggle by switching CSS variables.
- Replace mock data in `data.js` with dynamic content.
