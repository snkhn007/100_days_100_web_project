# Interview Prep Hub

A comprehensive, interactive web application to help you prepare for job interviews!

## Features

### 1. Interview Question of the Day
- 📅 Random daily interview question
- 📝 Shows detailed answer when "Show Answer" is clicked
- 💾 Uses localStorage to keep the same question all day
- 12 predefined questions covering various topics

### 2. Progress Tracker
- ✅ Track completion of technical topics
- 📊 Visual progress bar showing completion percentage
- 💾 Saves progress to localStorage across browser sessions
- Completed topics have distinct green styling

### 3. STAR Answer Builder
- ✨ Interactive tool to build STAR-format behavioral answers
- Real-time preview as you type
- 📋 Copy answer to clipboard with one click
- Clear form button to reset inputs
