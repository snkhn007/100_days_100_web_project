# 🎱 Magic 8 Ball

A fun and interactive Magic 8 Ball web app built using **HTML, CSS, and JavaScript**.  
Ask a yes/no question, shake the ball, and reveal your fortune with animated responses and sound effects.

---

## ✨ Features

- 🎱 Interactive Magic 8 Ball UI
- 🔮 Random fortune generation
- 🎵 Sound effects with mute/unmute toggle
- 📜 Real Magic 8 Ball responses
- 📱 Responsive modern design
- ✨ Smooth animations and glowing effects
- 🧹 Clear/reset functionality
- ⌨️ Supports both button click and Enter key input

## 🎮 How to Use

1. Type a yes/no question into the input field
2. Click **"Ask 8Ball"** or tap the Magic 8 Ball
3. Watch the ball shake and reveal your answer
4. View recent answers below the app

## 🎯 Possible Answers

### ✅ Affirmative Answers

- Outlook good

- Reply hazy, try again
- Ask again later
- Better not tell you now
- Cannot predict now
- Concentrate and ask again

### ❌ Negative Answers
- Don't count on it
- My reply is no

- Outlook not so good
- Very doubtful
