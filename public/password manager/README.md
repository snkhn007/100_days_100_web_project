# 🔐 Password Manager

## 🚀 Overview
Password Manager is a responsive web project designed to securely store and manage user credentials.  
Built with **HTML, CSS, and JavaScript**, it demonstrates how to implement authentication, password storage, and a clean UI for managing sensitive data.  
This project highlights practical front‑end techniques combined with security‑focused logic.

---

## ✨ Features
- ✅ Responsive layout for desktop, tablet, and mobile  
- ✅ Secure password storage using local/session storage  
- ✅ Add, view, and delete saved credentials  
- ✅ Dark/Light theme toggle for modern UI/UX  
- ✅ Navbar, header, and footer for structured layout  
- ✅ Beginner‑friendly implementation with modular code  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (for dynamic logic and storage)

## 🎮 Features Overview
### 🔑 Credential Management
- Add new credentials with username and password fields.  
- View saved credentials in a structured list.  
- Delete credentials securely when no longer needed.

- Toggle button updates dynamically with icons (☀️ / 🌙).
