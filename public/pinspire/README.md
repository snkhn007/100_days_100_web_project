# 📌 Pinspire – Discover & Save Ideas

Pinspire is a modern Pinterest-inspired web application that allows users to explore, search, save, and organize creative ideas across different categories such as travel, food, technology, art, fashion, and more.

Built using **HTML, CSS, and JavaScript**, Pinspire provides an interactive and visually appealing masonry-style gallery with detailed pin previews, user interactions, and a responsive design.

---

## 🚀 Features

### 🏠 Beautiful Home Feed

* Pinterest-style grid layout for displaying inspiration cards.
* High-quality image-based pins.
* Smooth hover effects and interactive UI.

### 🔍 Smart Search & Filtering

* Search pins by:

  * Title
  * Creator name
  * Categories

* Filter ideas using category pills:

  * Travel
  * Food
  * Architecture
  * Nature
  * Fashion
  * Interior
  * Art
  * Fitness
  * Technology
  * Books

### 📌 Save Your Favorite Pins

* Save and unsave pins with a single click.
* Dynamic save button states.
* Personalized interaction experience.

### 🖼️ Detailed Pin Modal

Clicking a pin opens a rich modal displaying:

* Large image preview.
* Pin title and description.
* Creator information.
* Follower count.
* Source website details.
* Save and share actions.

### 👥 Social Interactions

* Follow or unfollow creators.
* View existing comments.
* Add new comments.
* Real-time comment counter updates.
* Interactive notifications using toast messages.

### 🔔 User Experience Enhancements

* Responsive navigation bar.
* Search suggestions and category navigation.
* Notification and message placeholders.
* Keyboard support:

  * Close modal using the **Escape** key.
* Smooth toast alerts for user feedback.
