# 🎨 Pixel Art Maker

Pixel Art Maker is a simple and interactive web application that allows users to create pixel-based artwork directly in the browser. Users can customize the grid size, choose colors, draw freely, erase pixels, and clear the canvas to start a new creation.

Built using **HTML, CSS, and JavaScript**, this project provides a fun and beginner-friendly digital drawing experience.

---

## 🚀 Features

### 🖌️ Interactive Pixel Canvas

* Draw pixel art by clicking and dragging across the grid.
* Smooth mouse-based drawing experience.
* Individual pixel coloring for detailed designs.

### 📐 Customizable Grid Size

* Adjust the width and height of the canvas.
* Generate grids ranging from **1 × 1** to **64 × 64** pixels.
* Regenerate the canvas instantly with new dimensions.

### 🎨 Color Selection

* Built-in color picker to select any drawing color.
* Real-time color changes while creating artwork.

### 🧽 Eraser Tool

* Toggle rubber mode to erase individual pixels.
* Visual indication when the eraser is active.
* Automatically switches back to drawing mode when a new color is selected.

### 🗑️ Clear Canvas

* Remove all artwork with a single click.
* Quickly restart your drawing without changing the grid size.

### 🖥️ Responsive User Interface

* Clean and modern design.
* Scrollable canvas support for larger grids.
* Mobile-friendly and desktop-friendly layout.

## 🎯 How to Use

1. Enter the desired width and height of the pixel grid.
2. Click **Regenerate Grid** to create a new canvas.
3. Select a color using the color picker.
4. Click and drag on the grid to start drawing.
5. Enable the **Rubber** tool to erase pixels.
6. Use **Clear Canvas** to remove all drawings.
