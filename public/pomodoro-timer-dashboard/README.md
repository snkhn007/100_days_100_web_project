# 🎯 Focus Dashboard

A modern and responsive **Pomodoro Timer Dashboard** built with **HTML, CSS, and JavaScript**. The application combines a productivity-focused Pomodoro timer with an immersive ambient sound mixer, helping users maintain focus, reduce distractions, and improve work efficiency.

---

## 📖 Overview

Focus Dashboard is designed for students, developers, remote workers, and productivity enthusiasts who want a distraction-free environment for deep work sessions.

The dashboard provides:

* A customizable Pomodoro timer with multiple session modes
* Visual circular progress tracking
* Ambient sound generation using the Web Audio API
* Independent volume controls for each sound channel
* Modern dark-themed user interface
* Responsive layout for desktop and mobile devices
