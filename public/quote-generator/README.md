# Quote Generator

A simple and interactive Quote Generator built with **HTML**, **CSS**, and **JavaScript**. Generate inspirational quotes instantly with the click of a button.

## ✨ Features

* Random quote generation
* Clean and responsive user interface
* Lightweight and beginner-friendly project
* Accessible design using ARIA attributes
* No external dependencies required

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project/tree/Main/public/quote-generator.git
```

### 2. Navigate to the Project Folder

```bash
cd quote-generator
```

### 3. Open the Application

Simply open `index.html` in your preferred web browser.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📖 How It Works

1. The application stores a collection of quotes in a JavaScript array.
2. When the **New Quote** button is clicked, a random quote is selected.
3. The selected quote is displayed dynamically on the page.
