# 🏎️ Turbo Dash – 2D Arcade Car Racing Game

Turbo Dash is a fast-paced 2D arcade racing game built using **HTML5 Canvas, CSS3, and JavaScript**. Players control a high-speed car, dodge traffic, perform near misses to build combo multipliers, collect power-ups, and use turbo boosts to achieve the highest score.

The game features a futuristic neon-inspired interface, smooth animations, mobile controls, and dynamic gameplay designed for an engaging browser-based racing experience.

---

## 🚀 Features

### 🚗 Arcade Racing Gameplay

* Smooth left and right car movement.
* Endless racing challenge with increasing difficulty.
* Responsive controls for desktop and mobile devices.
* Real-time score tracking.

### ⚡ Turbo Boost System

* Activate a temporary speed boost.
* Animated boost meter to show available energy.
* Special visual effects during boost mode.

### 🔥 Combo & Near-Miss System

* Earn bonus points by narrowly avoiding obstacles.
* Build combo multipliers for higher scores.
* Track maximum combo streaks.

### 🛡️ Power-Up Collection

The game includes multiple power-ups:

* 🛡️ **Shield** – Protects the player from one collision.
* 🐌 **Slow Motion** – Reduces game speed temporarily.
* ✖️ **Score Multiplier** – Doubles score rewards.

### 🌙 Day & Night Mode

* Toggle between day and night themes.
* Dynamic visual changes for a different racing atmosphere.

### 📊 Advanced Game HUD

The heads-up display provides:

* Current score.
* Highest score record.
* Active combo multiplier.
* Boost energy status.
* Active power-up indicators.

### 💥 Game Over Statistics

After every race, players can view:

* Final score.
* Best score.
* Total boosts used.
* Number of near misses.
* Highest combo achieved.

### 📱 Mobile-Friendly Controls

Touch controls are available for mobile devices:

* ◀ Move Left
* ▶ Move Right
* ⚡ Activate Boost

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/100_days_100_project.git
```

### 2. Navigate to the Project Directory

```bash
cd turbo-dash
```

### 3. Run the Game

Open `index.html` directly in a modern web browser.

For the best development experience, use a local development server such as **VS Code Live Server**.
