# ⚡ Reaction Time Tester - Ultimate Reflex Challenge

A modern and interactive reaction-based web game inspired by F1 driver reflex training. Test your speed, memory, accuracy, and focus across multiple exciting game modes while tracking your performance with leaderboards, achievements, and detailed statistics.

---

# 🎮 Game Modes

## 🚦 Classic F1 Reaction

* Wait for the traffic lights to turn green and click instantly
* Test raw reaction speed
* Millisecond precision tracking
* F1-inspired gameplay

---

## 🎨 Color Match Challenge

* React only when the correct target color appears
* Random color generation
* Focus and reflex testing
* Instant timing feedback

---

## 🎯 Moving Target Challenge

* Hit randomly moving targets quickly
* Multi-round scoring system
* Average reaction tracking
* Accuracy-based gameplay

---

## 🧠 Memory Reflex Game

* Memorize flashing patterns
* Repeat sequence correctly
* Memory + reflex combination
* Interactive visual feedback
