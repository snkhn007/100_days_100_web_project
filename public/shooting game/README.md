## Target Shooter

A simple browser-based shooting game where the player aims to hit a target to score points within a limited time. The closer to the center, the higher the score.

## Features
- Interactive target with multiple scoring rings
- Real-time score tracking system
- High score storage display
- Countdown timer-based gameplay (15 seconds)
- Start screen with game instructions
- Pause and restart controls
- Animated gun and shooting effects
- Score popup feedback on hits
- Visual glowing background for immersive gameplay

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run

- Download or clone the project folder
- Open index.html in any modern web browser (Chrome, Edge, Firefox, etc.)
- Click PLAY GAME to start
- Use mouse interaction to aim and shoot the target
- Try to score as high as possible before time runs out
