# 🐍 Snake Game

A modern, feature-rich **Snake Game** built using **HTML5 Canvas, CSS3, and Vanilla JavaScript**. This project reimagines the classic arcade snake experience with smooth animations, customizable themes, sound effects, mobile controls, achievements, and advanced game mechanics.

---

## 🎮 Features

### Core Gameplay

* 🐍 Classic snake movement and growth mechanics
* 🍎 Animated food system with glowing visual effects
* ⚡ Progressive difficulty with level-based speed increase
* 💀 Collision detection for walls and snake body
* 🏆 Score tracking with high score display
* 🎯 Real-time level progression

### Visual & User Experience

* 🎨 Dark and light theme support
* 🌈 Customizable snake color picker with saved preferences
* ✨ Smooth animations powered by `requestAnimationFrame`
* 🖥️ Responsive UI optimized for desktop and mobile devices
* 🔥 Retro arcade-inspired interface and visual effects
* ⏸️ Pause and resume functionality

### Controls

#### Desktop

| Key   | Action         |
| ----- | -------------- |
| W / ↑ | Move Up        |
| A / ← | Move Left      |
| S / ↓ | Move Down      |
| D / → | Move Right     |
| P     | Pause / Resume |

#### Mobile

* Touch D-pad controls
* Swipe gestures directly on the game canvas
* Haptic vibration feedback (supported devices)

## 🚀 Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project.git
```

2. Navigate to the project directory:

```bash
cd snake-game
```

3. Open `index.html` in your browser.

No additional dependencies or installations are required.

---

## 🎯 How to Play

1. Press **Start Game**.
2. Control the snake using keyboard keys or mobile controls.
3. Eat food to increase your score and grow longer.
4. Every few food items increase your level and make the game faster.
5. Avoid hitting the walls or your own body.
6. Try to achieve the highest score possible.
