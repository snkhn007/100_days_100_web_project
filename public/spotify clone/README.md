# Spotify Clone 🎵

A responsive and visually appealing **Spotify Clone** built using **HTML, CSS, and JavaScript**.
This project recreates the core layout and user interface of the popular music streaming platform while focusing on responsive design, interactive controls, and smooth user experience.

---

## 🚀 Features

* 🎧 Modern Spotify-inspired UI
* 📱 Fully responsive design
* ▶️ Play/Pause music functionality
* ⏭️ Next and Previous track controls
* 🎵 Dynamic song list display
* 🎚️ Interactive progress bar
* 📀 Playlist and album sections
* 🌙 Clean and minimal interface

---

| Technology       | Purpose                       |
| ---------------- | ----------------------------- |
| HTML5            | Structure                     |

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/spotify-clone.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd spotify-clone
```

### 3️⃣ Open the Project

Open the `index.html` file directly in your browser.

### Optional: Run with Live Server

If you're using VS Code:

```bash
Right Click → Open with Live Server
```

## 🎮 Usage

* Click the **Play** button to start music playback
* Use **Next** and **Previous** controls to switch tracks
* Interact with the progress bar during playback
* Explore playlists and album sections
