# Spotify Clone

A Spotify web player clone featuring a music library with playlist cards.

## Features

- Music Library - Browse through various playlists
- Modern UI - Clean Spotify-inspired dark theme interface
- Playlist Cards - Visual playlist representation with cover images
- Responsive Design - Works on desktop and tablet devices
- Sidebar Navigation - Home, Search, Your Library

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Project Structure

- index.html
- images/ folder with all playlist covers
- README.md
