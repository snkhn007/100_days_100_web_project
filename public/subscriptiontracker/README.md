# Subscription Tracker

## Overview
A web application that helps users manage and monitor their subscriptions in one place.

## Features
- Add subscriptions
- Edit subscriptions
- Delete subscriptions
- Track renewal dates
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Installation
1. Clone the repository.
2. Navigate to the project folder.
3. Open `index.html` in your browser.

## Usage
- Add subscription details.
- Manage existing subscriptions.
- Track upcoming renewals.
