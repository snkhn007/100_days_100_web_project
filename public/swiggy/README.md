# Swiggy Instamart Clone

A front-end clone of Swiggy Instamart built using HTML, CSS, and JavaScript.

## Features

- Product category showcase
- Search bar UI
- Customer review section
- Dynamic comment posting
- Comment deletion
- Responsive layout
- Footer with category navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
