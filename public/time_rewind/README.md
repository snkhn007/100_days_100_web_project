# ⏳ CHRONO ECHO

A browser-based time manipulation puzzle game where every second matters. Navigate challenging levels, master temporal mechanics, and use limited rewinds strategically to overcome obstacles and complete objectives.

## 🎮 Overview

CHRONO ECHO is a 2D puzzle-action game built with HTML5 Canvas and JavaScript. Players progress through increasingly difficult levels while managing time, solving puzzles, and utilizing a limited rewind system to correct mistakes and discover optimal solutions.

The core gameplay revolves around controlling time itself, creating a unique challenge that rewards planning, experimentation, and precision.

## 🎯 Gameplay Strategy

### Use Rewinds Wisely

Rewinds are limited. Avoid using them unnecessarily early in a level.

### Learn Level Patterns

Observe obstacles and puzzle mechanics before committing to actions.

### Optimize Your Runs

Replay completed levels to improve your time and reduce rewind usage.

### Experiment

Sometimes the best solution requires testing different approaches and learning from mistakes.
